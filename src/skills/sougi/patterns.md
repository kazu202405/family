## 4. よくある状況と対応パターン

### パターン1「親がまだ元気だけど葬儀のことを考えておきたい」

**相談者の気持ち**: 備えたいが、不謹慎ではないかという不安がある

**対応の流れ**:

1. 「大切な方を思うからこそ、備えておこうとされているのですね」と気持ちを受け止める
2. 事前準備のメリットを伝える（本人の希望を尊重できる、費用を比較検討できる、いざというとき慌てずに済む）
3. まずはエンディングノートの活用を提案する
4. 葬儀社への事前相談（無料）を勧める
5. 重要書類の保管場所の共有を提案する

### パターン2「急に亡くなって何をすればいいかわからない」

**相談者の気持ち**: 混乱と悲しみの中、何から手をつければいいかわからない

**対応の流れ**:

1. 「突然のことで、おつらいですね。よくご相談くださいました」と最初に寄り添う
2. 「今すぐやるべきことは限られています。一つずつ確認しましょう」と安心感を提供する
3. 最初の24時間でやることを簡潔に伝える
   - 死亡診断書の受け取り
   - 葬儀社への連絡
   - 近親者への連絡
4. 「今日はここまでで大丈夫です」と区切りを示す
5. 手続きの全体像は、気持ちが少し落ち着いてからお伝えする

### パターン3「葬儀社の見積もりが適正かわからない」

**相談者の気持ち**: 費用が妥当なのか判断できず不安

**対応の流れ**:

1. 複数社の見積もりを比較することが最も重要であることを伝える
2. 見積もりの確認ポイントを伝える
   - 基本料金に含まれるものと含まれないものの区別
   - 追加料金が発生する条件
   - 飲食・返礼品の費用が別途かどうか
   - 僧侶へのお布施が含まれるかどうか
3. 総額だけでなく内訳を確認するよう勧める
4. 不明な項目は遠慮なく質問してよいと伝える

### パターン4「兄弟で葬儀のやり方について意見が合わない」

**相談者の気持ち**: 家族間の意見の違いに悩み、板挟みになっている

**対応の流れ**:

1. 「ご家族の中で意見が分かれるのは、それだけ皆さんが故人のことを大切に思っている証拠です」と受け止める
2. 合意形成のポイントを伝える
   - 故人の生前の希望を最優先する
   - エンディングノートや普段の会話の中での発言を思い出す
   - 費用負担の方法を明確にする
   - 第三者（葬儀社のスタッフ、僧侶）の意見を参考にする
3. 全員が100%満足でなくても「故人のために」という共通の軸で歩み寄ることを提案する

### パターン5「墓じまいしたいが親戚の反対がある」

**相談者の気持ち**: 現実的な必要性と親戚への配慮の間で悩んでいる

**対応の流れ**:

1. 「墓じまいはとても大きな決断ですね。お一人で抱え込まないでください」と共感する
2. 反対する方の気持ちにも理解を示す（先祖供養への思い、変化への抵抗）
3. 説得ではなく対話のポイントを伝える
   - 墓じまいが必要な理由（管理者の不在、高齢化、遠方）を具体的に共有する
   - 墓じまい後の供養方法（永代供養、納骨堂など）を提案する
   - 「供養の心は変わらない」ことを伝える
4. 全員での現地訪問や合同法要を提案し、気持ちの整理を促す

### パターン6「悲しくて手続きが進められない」

**相談者の気持ち**: 悲しみで身動きが取れない。手続きの存在自体が負担

**対応の流れ**:

1. 「悲しいお気持ちの中で手続きのことまで考えるのは、本当に大変なことです」と受け止める
2. 「悲しみは自然な感情です。無理に乗り越えようとしなくて大丈夫です」とグリーフケアの視点を伝える
3. 最低限必要な期限付きの手続きだけを伝える（すべてを一度に伝えない）
4. 代行・サポートの選択肢を伝える
   - 家族や親族に手続きを分担してもらう
   - 司法書士や行政書士に手続きを代行してもらう
   - 市区町村の「おくやみコーナー」を利用する（設置している自治体の場合）
5. 心身の不調が続く場合は、グリーフケアの専門家や心療内科への相談を勧める
