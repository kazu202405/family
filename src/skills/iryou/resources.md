## 5. 相談先ガイド

### 5-1. 医療ソーシャルワーカー（MSW）── 病院内の相談窓口

- **場所**: 病院内の「地域連携室」「患者支援センター」「医療福祉相談室」などの名称
- **対応内容**: 医療費の不安、退院後の生活、転院先の相談、介護保険の手続き支援、家族の精神的なサポート
- **利用方法**: 病棟の看護師や受付に「相談室を利用したい」と伝える。入院中であれば無料で利用できる
- **活用のコツ**: 入院早期に一度つながっておくと、退院準備がスムーズに進む。遠方の家族は電話で相談も可能である

### 5-2. 地域包括支援センター ── 退院後の生活の相談

- **場所**: 各市区町村に設置されている。中学校区に1カ所程度の目安で配置されている
- **対応内容**: 介護保険の申請支援、ケアマネジャーの紹介、退院後の生活全般の相談、権利擁護
- **利用方法**: 電話または窓口で相談する。利用料は無料である
- **受付時間**: 一般的に平日8:30〜17:00が多い（自治体により異なる場合がある。土日祝は閉庁の場合が多い）
- **対象**: 65歳以上の高齢者とその家族。介護保険の認定を受けていなくても相談できる
- **探し方**: 市区町村の公式サイトで検索するか、市区町村の高齢者福祉担当課に電話で問い合わせる

### 5-3. かかりつけ医

- **役割**: 日常的な健康管理、専門医への紹介、入退院時の情報共有、介護保険の主治医意見書の作成
- **探し方**: 自宅近くの診療所を受診する、地域の医師会ウェブサイトで検索する、地域包括支援センターに相談する
- **活用のコツ**: 親の既往歴や服薬情報を整理しておくと、スムーズにかかりつけ医を見つけやすい

### 5-4. 在宅療養支援診療所

- **役割**: 24時間対応の在宅医療を提供する診療所である。訪問診療を中心に、緊急時の往診や看取りにも対応する
- **対象**: 通院が困難で、自宅での療養を希望する患者
- **探し方**: かかりつけ医やMSWに紹介を依頼する。各都道府県の医療情報ネット（医療機能情報提供制度）で検索も可能である
- **費用**: 医療保険が適用される。月2回の訪問診療の場合、1割負担で月額6,000〜7,000円程度が目安である（状態や処置内容で変動する）

### 5-5. 主な電話相談窓口

| 窓口名 | 電話番号 | 受付時間 |
|---|---|---|
| よりそいホットライン（一般社団法人 社会的包摂サポートセンター） | 0120-279-338 | 24時間対応（通話無料） |
| 地域包括支援センター | 市区町村により異なる | 平日8:30〜17:00が一般的 |
| 各自治体の高齢者相談窓口 | 市区町村により異なる | 平日8:30〜17:15が一般的 |
| 認知症の人と家族の会 電話相談 | 0120-294-456 | 月〜金 10:00〜15:00（祝日・年末年始を除く） |

### 5-6. 各窓口の使い分け

| 困りごと | まず相談する先 |
|---|---|
| 入院中の不安（費用、退院後の生活） | 医療ソーシャルワーカー（MSW） |
| 介護保険の申請・ケアマネ探し | 地域包括支援センター |
| 日常的な体調の変化・健康相談 | かかりつけ医 |
| 通院できない・自宅で療養したい | 在宅療養支援診療所 |
| 認知症の診断を受けたい | かかりつけ医 → 物忘れ外来・認知症疾患医療センター |
| 退院直後の医療処置が必要 | 訪問看護ステーション（MSWまたはケアマネ経由で手配） |

## 6. 家族への伝え方

### 6-1. 親に受診を勧める言い方

**避けるべき言い方**:
- 「最近おかしいから病院に行って」（本人のプライドを傷つける）
- 「認知症かもしれないよ」（不安をあおり、拒否が強まる）
- 「みんな心配しているんだから」（プレッシャーを与える）

**効果的な言い方の例**:
- 「健康診断の時期だから、一緒に行かない？」
- 「最近よく眠れてる？ 先生に相談してみたらどうかな」
- 「かかりつけの先生にちょっと聞いてみよう。薬のこともあるし」
- 「元気でいてほしいから、定期的に診てもらえると安心なんだけど」

**ポイント**: 「病気を見つけに行く」のではなく「元気を確認しに行く」というフレーミングが有効である。

### 6-2. 延命治療・ACP（人生会議）の切り出し方

**いきなり「延命治療どうする？」と聞かない**。重い話題を唐突に出すと、本人も家族も構えてしまう。

**切り出し方の例**:
- テレビや新聞で関連する話題が出たときに「うちはどうしたいか、考えたことある？」
- 知人や親戚の入院・看取りの話が出たときに「もしそうなったら、どうしてほしい？」
- 「最近"人生会議"っていうのがあるみたいだよ。家族で話しておくといいらしい」

**話し合いで大切なこと**:
- 結論を急がない。「今日は聞いてみただけ」で十分である
- 本人の気持ちが変わることは自然なことだと伝える
- 話し合った内容はメモに残しておく（日付も記録する）
- 家族全員が同じ情報を共有できるようにする

### 6-3. 入院中の家族間での情報共有方法

入院が長引くと、家族間で情報の伝達漏れやすれ違いが起きやすい。

**具体的な方法**:
- **家族グループ（LINE等）の作成**: 面会した人が当日の様子を共有する
- **共有ノートの活用**: 病室に1冊ノートを置き、面会者が日付と様子を記録する
- **役割分担の明確化**: 「病院との連絡窓口」「保険手続き担当」「面会ローテーション」など
- **定期的な家族会議**: 週に1回程度、状況と今後の方針を話し合う時間をつくる
- **MSWの説明に家族代表が同席**: 聞いた内容を他の家族に共有する

**AIカウンセラーの対応**: 「ご家族で情報を共有する仕組みをつくっておくと、お互いの負担が減りますし、認識のずれも防げます」と提案する。

## 7. 制度・法的情報

### 7-1. 高額療養費制度

医療費の自己負担額が一定額（自己負担限度額）を超えた場合、超えた分が後から払い戻される制度である。

**自己負担限度額の目安（70歳以上・一般所得の場合）**:

| 区分 | 外来（個人） | 入院（世帯） |
|---|---|---|
| 現役並み所得III（課税所得690万円以上） | 252,600円＋(医療費-842,000円)×1% | 同左 |
| 現役並み所得II（課税所得380万円以上） | 167,400円＋(医療費-558,000円)×1% | 同左 |
| 現役並み所得I（課税所得145万円以上） | 80,100円＋(医療費-267,000円)×1% | 同左 |
| 一般（課税所得145万円未満） | 18,000円（年間上限144,000円） | 57,600円 |
| 住民税非課税世帯II | 8,000円 | 24,600円 |
| 住民税非課税世帯I | 8,000円 | 15,000円 |

**注意**: 上記は概算の目安である。正確な限度額は加入する健康保険に確認すること。食事代・差額ベッド代・先進医療費などは対象外である。

### 7-2. 限度額適用認定証の申請方法

入院前または入院中に「限度額適用認定証」を取得しておくと、窓口での支払いが自己負担限度額までに抑えられる。

**申請手順**:
1. 加入している健康保険の窓口に申請する
   - **国民健康保険**: 市区町村の国保窓口
   - **後期高齢者医療**: 市区町村の後期高齢者医療担当窓口
   - **健康保険組合・協会けんぽ**: 各保険者に直接申請
2. 申請書を提出する（郵送も可能な場合が多い）
3. 認定証が届いたら、病院の窓口に提示する

**ポイント**:
- 入院がわかった時点で早めに申請するのが望ましい
- マイナ保険証を利用している場合、認定証がなくても窓口で限度額が適用される医療機関が増えている（対応状況は病院に確認）。なお、2024年12月に従来の健康保険証の新規発行は終了しており、マイナ保険証への移行が進んでいる
- 認定証を提示しなかった場合でも、後から高額療養費として払い戻し申請ができる

### 7-3. 医療費控除

1年間（1月〜12月）に支払った医療費の合計が一定額を超えた場合、確定申告で税金の還付を受けられる制度である。

**対象となる費用の例**:
- 入院費・手術費の自己負担分
- 通院のための交通費（公共交通機関）
- 処方薬の費用
- 訪問看護・訪問リハビリの自己負担分
- 介護保険サービスの一部（医療系サービス）

**対象外の例**:
- 差額ベッド代（自己都合の場合）
- 入院中の身の回り品の購入費
- 自家用車での通院にかかるガソリン代・駐車場代

**控除の計算**:
- 医療費控除額 = 支払った医療費 - 保険等で補てんされた金額 - 10万円（※）
- ※総所得金額が200万円未満の場合は、総所得金額の5%

**セルフメディケーション税制**: 健康の維持増進および疾病予防への取組として一定の取組を行う個人が、対象の市販薬（スイッチOTC医薬品）の購入費が年間12,000円を超えた場合に適用可能である（上限88,000円）。医療費控除との選択制であり、いずれか一方のみ申告できる。

**手続き**: 翌年の確定申告で申告する。医療費の領収書は5年間保管が必要である。

## 8. カウンセラー向けチェックリスト

本セクションは、AIカウンセラーが相談対応時に確認すべき項目をまとめたものである。

### 8-1. 相談開始時の確認事項

- [ ] 相談者の状況を把握したか（入院前・入院中・退院後のいずれか）
- [ ] 相談者の感情状態を確認したか（パニック状態、自責、不安、混乱など）
- [ ] 緊急性の高い事項がないか確認したか（容体の急変、医療的緊急事態への言及など）
- [ ] 医療判断に踏み込む発言をしていないか

### 8-2. 情報提供時の確認事項

- [ ] 提示条件に合致する情報のみを提供しているか
- [ ] 相談者の状況に応じた優先順位で案内しているか
- [ ] 専門窓口への橋渡しを適切に行っているか（MSW、地域包括支援センター等）
- [ ] 制度の説明は一般的な情報にとどめ、個別の判断を行っていないか

### 8-3. 相談終了時の確認事項

- [ ] 相談者が次に取るべきアクションを明確に伝えたか
- [ ] 相談できる窓口（電話番号・場所）を案内したか
- [ ] 相談者の不安が軽減されたか、またはさらなるサポートが必要かを確認したか
- [ ] 「一人で抱え込まなくて大丈夫です」という安心感を伝えたか

### 8-4. 禁止行為の最終チェック

- [ ] 症状の診断・治療法の推奨・薬の効果に関する発言をしていないか
- [ ] 禁止表現（セクション2-4参照）を使用していないか
- [ ] 恐怖や不安を煽る表現を使っていないか
- [ ] 根拠のない楽観的発言（「大丈夫ですよ」「きっと治ります」等）をしていないか
