## 3. 基礎知識

### 3-1. かかりつけ医の見つけ方と役割

▶ 提示条件：相談者が通院先を探している、またはかかりつけ医がいないと述べた場合に案内する

**かかりつけ医とは**: 日常的な健康管理や病気の初期対応を担当する、身近な医師のことである。

**見つけ方**:
- 自宅から通いやすい距離にある診療所・クリニックを探す
- 地域の医師会ウェブサイトで検索できる場合がある
- 地域包括支援センターに相談すると、地域の医療機関情報を教えてもらえる
- 複数の持病がある場合は、内科を中心に相談できる医師が望ましい

**かかりつけ医の主な役割**:
- 日常的な体調管理と健康相談
- 必要に応じた専門医への紹介状の作成
- 入退院時の情報共有と在宅医療への橋渡し
- 介護保険の主治医意見書の作成

### 3-2. 認知症の受診先

▶ 提示条件：認知症の受診について質問された場合に受診先の選択肢を案内する

認知症が疑われるとき、どこを受診すればよいかわからないという相談は非常に多い。

**ステップ1: かかりつけ医に相談する**
- まずは日頃の様子を伝え、認知機能の簡易検査を受けられる
- かかりつけ医から専門医への紹介状を書いてもらえる

**ステップ2: 専門医を受診する**
- **物忘れ外来**: 認知症の診断に特化した外来。総合病院や大学病院に設置されていることが多い
- **認知症疾患医療センター**: 都道府県が指定する専門医療機関。診断から対応まで包括的に行う
- **精神科・神経内科（脳神経内科）**: 認知症の専門的な診断・治療を行う

**受診時に持参すると役立つもの**:
- 気になる症状のメモ（いつから、どんな場面で）
- お薬手帳
- 保険証、介護保険証（あれば）
- 本人の日常を知る家族の同行

### 3-3. 入院時にやることチェックリスト

▶ 提示条件：相談者が入院直後であると明言した場合にチェックリストを提示する

突然の入院でも、以下のリストを順に確認すれば対応できる。

**入院手続き関連**:
- [ ] 入院申込書の記入・提出
- [ ] 保証人（連帯保証人）の確認と署名
- [ ] 保険証の提示（健康保険証、後期高齢者医療被保険者証）
- [ ] 限度額適用認定証の申請（後述）── 入院がわかった時点で早めに手続きする
- [ ] 介護保険証がある場合は病院に伝える

**持ち物の準備**:
- [ ] 着替え（前開きの肌着・パジャマが便利）
- [ ] 洗面用具（歯ブラシ、タオル、ティッシュ）
- [ ] お薬手帳と現在服用中の薬
- [ ] 必要最低限の現金
- [ ] 補聴器、眼鏡、入れ歯ケースなど普段使用しているもの
- [ ] 靴（かかとのある、滑りにくいもの）

**確認しておくこと**:
- [ ] 面会時間とルール
- [ ] 担当医・担当看護師の名前
- [ ] 医療ソーシャルワーカー（相談室）の場所と連絡先
- [ ] 入院中のケアマネジャーへの連絡（介護サービスを利用している場合）

### 3-4. 医療ソーシャルワーカー（MSW）の役割と相談の仕方

▶ 提示条件：退院が迫っている場合にMSWへの相談を優先的に案内する。また、医療費や退院後の生活について不安を訴えた場合にも提示する

**MSWとは**: 病院に所属する福祉の専門職である。医療費、退院後の生活、転院先の相談など、治療以外の困りごとに幅広く対応する。

**相談できること**:
- 医療費の支払いに関する不安
- 退院後の生活の準備（介護サービスの調整、施設探し）
- 転院先の情報提供
- 社会保障制度の案内（高額療養費、障害年金など）
- 家族の精神的な負担に対する相談

**相談の仕方**:
- 病院の「地域連携室」「患者支援センター」「医療相談室」などの名称で設置されている
- 看護師や受付に「ソーシャルワーカーに相談したい」と伝えれば案内してもらえる
- 入院早期に一度つながっておくと、退院に向けた準備がスムーズに進む

### 3-5. 退院支援の流れ

▶ 提示条件：退院の見通しが出た、または退院後の生活に不安があると相談された場合に提示する

入院後、病状が安定してくると退院に向けた調整が始まる。

**退院支援の主なステップ**:

1. **退院の方向性の確認**: 主治医から退院の見通しが伝えられる
2. **退院前カンファレンス**: 病院スタッフ（医師、看護師、リハビリ職、MSW）と、在宅側のスタッフ（ケアマネジャー、訪問看護師など）が集まり、退院後の生活を話し合う
3. **在宅環境の調整**: 必要に応じて介護サービスの導入、住宅改修、福祉用具のレンタルを手配する
4. **退院日の決定と退院指導**: 退院後の注意点（服薬、食事、リハビリ、通院予定）の説明を受ける

**退院先の主な選択肢**:
- **自宅**: 介護サービスを組み合わせて在宅で生活する
- **回復期リハビリテーション病院**: リハビリを集中的に行うための転院
- **介護老人保健施設（老健）**: 在宅復帰を目指す中間的な施設
- **介護施設（特養、有料老人ホームなど）**: 在宅が困難な場合の生活の場

### 3-6. 在宅医療の種類

▶ 提示条件：退院後の自宅療養について質問があった場合、または通院困難な状況が語られた場合に提示する

退院後に自宅で療養を続ける場合、以下のサービスを利用できる。

**訪問診療**:
- 医師が定期的に自宅を訪問し、診察・処方を行う
- 通院が難しい患者が対象である
- 急変時の電話相談や緊急往診に対応する診療所もある

**訪問看護**:
- 看護師が自宅を訪問し、健康状態の観察、医療処置、服薬管理などを行う
- 医師の指示書に基づいて実施される
- 医療保険・介護保険のどちらでも利用可能である（状況による）

**訪問リハビリテーション**:
- 理学療法士・作業療法士・言語聴覚士が自宅を訪問し、機能回復訓練を行う
- 退院後の身体機能維持・回復を支援する

### 3-7. セカンドオピニオンの取り方

▶ 提示条件：治療方針への疑問や不安が語られた場合、またはセカンドオピニオンについて直接質問された場合に提示する

主治医の診断や治療方針に納得がいかない場合、別の医師の意見を聞くことができる。

**手順**:
1. 主治医にセカンドオピニオンを希望していることを伝える
2. 紹介状（診療情報提供書）と検査データを受け取る
3. セカンドオピニオン外来のある病院に予約を取る
4. 受診し、別の医師の見解を聞く
5. 結果を踏まえ、元の主治医と今後の方針を相談する

**知っておくとよいこと**:
- セカンドオピニオンは保険適用外（自費）の場合が多い（目安: 5,000〜30,000円程度）
- 主治医に言い出しにくい場合は、看護師やMSWに相談できる
- セカンドオピニオンは「主治医を変える」ことではなく「別の視点を得る」ことである
