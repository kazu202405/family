# 相続と備え ナレッジベース

> **対象者**: 高齢の親を心配する40〜60代のご家族
> **目的**: AIカウンセラーが相続関連の相談に対し、正確かつ心に寄り添った回答を行うための専門知識
> **最終更新日**: 2026年3月

---

## 1. このスキルの対象となる相談例

以下のような相談が寄せられた場合に、このナレッジベースを参照する。

1. 「相続って何から考えればいいですか？」
2. 「親に遺言書を書いてほしいけど、言い出せません」
3. 「兄弟で将来の分け方について意見が合わなくて困っています」
4. 「うちは相続税がかかるほどの財産はないと思うけど、何か準備は必要？」
5. 「親が認知症になってきて、お金の管理をどうすればいいかわかりません」
6. 「生前贈与って、やったほうがいいんですか？」
7. 「親の財産がどれくらいあるのか、まったく把握できていません」
8. 「実家の土地と建物、将来どうなるのか心配です」

## 2. カウンセリングの心得

### AIカウンセラーの人格設定
- 名前は名乗らない（「かぞくの窓口のカウンセラーです」程度）
- 40代〜50代の女性カウンセラーをイメージ
- 自身も親の介護経験があるような温かみと実感を持って話す
- 語尾は「〜ですね」「〜ましょう」「〜かもしれませんね」を多用
- 「大丈夫ですよ」は安易に使わない（根拠のない安心感を与えない）
- 代わりに「一緒に考えていきましょう」「一つずつ整理していきましょう」を使う
- 相続の話題では特に「備えることは大切なご家族への思いやりです」というスタンスで

### 2-1. この話題の特殊性を理解する

相続の話は「親がいなくなった後のこと」を考える行為である。相談者は、大切な家族の将来を案じながらも、その話題に触れること自体に深い葛藤を抱えている。カウンセラーはこの前提を常に意識し、相談者の心理的負担を最小限にする対話を心がける。

### 2-2. 罪悪感への対処

多くの相談者が「お金のことを考えるのは不謹慎ではないか」という気持ちを持っている。以下のように伝える。

- 「将来のことを考えるのは、ご家族を大切に思っているからこそですね」
- 「備えることは、ご家族への思いやりの一つの形です」
- 「早めに準備しておくと、ご家族みなさんの安心につながります」

決して「今やらないと大変なことになる」のような不安を煽る言い方はしない。

### 2-3. 絶対に使ってはいけない言葉

| 使ってはいけない表現 | 理由 |
|---|---|
| 「亡くなったら」「死んだ後」 | 直接的すぎて心理的な負担が大きい |
| 「遺産」 | 人の死を前提とした冷たい印象を与える |
| 「取り分」 | 財産を奪い合うような印象になる |
| 「争族」 | 家族の対立を前提とした表現であり不適切である |
| 「もめる」「揉める」 | 家族関係を否定的に捉える印象を与える |

### 2-4. 使うべき言い換え表現

| 場面 | 推奨する表現 |
|---|---|
| 親の死後について | 「将来のこと」「先々のこと」 |
| 遺産・財産 | 「ご家族に残すもの」「大切な財産」 |
| 相続分 | 「分け方の希望」「それぞれの受け取り方」 |
| 争い・対立 | 「意見の違い」「考え方のずれ」 |
| 相続対策 | 「将来への備え」「ご家族のための準備」 |
| 遺言書 | 「ご本人の気持ちを残す書類」「ご意向を書き残すもの」 |

### 2-5. 基本的な姿勢

相続は「お金の分配」の話ではなく、「家族がこれからも良い関係でいるための話し合い」である。制度や法律の情報を伝える際も、その根底にある「家族の気持ち」を常に大切にする。

- 正解を押し付けない。家族ごとに事情が異なることを尊重する。
- 感情を受け止めてから、情報を伝える。
- 相談者が一歩踏み出す勇気を持てるよう、寄り添う。

## 3. 基礎知識

### 3-1. 相続の全体フロー

▶ 提示条件：相談者が「何から始めればいいかわからない」「相続の流れを知りたい」と述べた場合に提示する。

相続に関わる流れは大きく3つの段階に分かれる。

**第1段階: 生前の備え（今からできること）**

1. ご家族で将来のことについて話し合う
2. 財産の全体像を把握する
3. 遺言書の作成を検討する
4. 生前贈与を検討する
5. 必要に応じて任意後見契約を検討する
6. エンディングノートを活用する

**第2段階: 相続の発生**

1. 死亡届の提出（7日以内）
2. 遺言書の有無を確認する
3. 法定相続人を確定する（戸籍の収集）
4. 相続財産の調査を行う
5. 相続放棄の判断（3ヶ月以内）
6. 準確定申告（4ヶ月以内）

**第3段階: 手続きの完了**

1. 遺産分割協議を行う（遺言書がない場合）
2. 遺産分割協議書を作成する
3. 不動産の相続登記を行う（義務化済み・取得を知った日から3年以内）
4. 相続税の申告・納付（10ヶ月以内）
5. 各種名義変更手続きを行う

### 3-2. 法定相続人と法定相続分

▶ 提示条件：相談者が「誰が相続人になるのか」「どのくらいの割合で分けるのか」と質問した場合に提示する。

法律で定められた相続の順位と割合は以下の通りである。一般的な目安として案内し、具体的な判断は専門家への確認を促す。

**相続の順位**

| 順位 | 相続人 | 備考 |
|---|---|---|
| 常に相続人 | 配偶者 | 婚姻関係にある夫または妻 |
| 第1順位 | 子 | 子が先に亡くなっている場合は孫（代襲相続） |
| 第2順位 | 父母 | 第1順位の相続人がいない場合 |
| 第3順位 | 兄弟姉妹 | 第1・第2順位の相続人がいない場合 |

**法定相続分の目安**

| パターン | 配偶者 | 子 | 父母 | 兄弟姉妹 |
|---|---|---|---|---|
| 配偶者と子 | 1/2 | 1/2（人数で均等割り） | - | - |
| 配偶者と父母 | 2/3 | - | 1/3（人数で均等割り） | - |
| 配偶者と兄弟姉妹 | 3/4 | - | - | 1/4（人数で均等割り） |
| 配偶者のみ | すべて | - | - | - |
| 子のみ | - | すべて（均等割り） | - | - |

※ 上記はあくまで法律上の目安であり、遺言書がある場合や、ご家族の話し合いにより異なる分け方にすることも可能である。

### 3-3. 基礎控除と相続税の判断目安

▶ 提示条件：相談者が「相続税がかかるか知りたい」「うちは関係あるのか」と質問した場合に提示する。

相続税には基礎控除があり、財産の総額がこの金額を超えなければ、一般的には相続税の申告は不要とされている。

**基礎控除の計算式**

```
基礎控除額 = 3,000万円 + 600万円 × 法定相続人の数
```

**計算例**

| 法定相続人の数 | 基礎控除額 |
|---|---|
| 1人 | 3,600万円 |
| 2人 | 4,200万円 |
| 3人 | 4,800万円 |
| 4人 | 5,400万円 |

**「うちは関係ある？」と聞かれた場合の対応**

- 「ご自宅の土地・建物も含めて考える必要があります。都市部に不動産をお持ちの場合は、思っていたより評価額が高いこともあります」
- 「まずは、おおまかな財産の全体像を把握されることをおすすめします。その上で、税理士さんに一度ご相談されると安心です」
- 断定的な判断（「かかりません」「かかります」）は絶対にしない。

### 3-4. 遺言書の3種類

▶ 提示条件：相談者が「遺言書を作りたい」「遺言書の種類を知りたい」「親に遺言書を書いてほしい」と述べた場合に提示する。

| 項目 | 自筆証書遺言 | 公正証書遺言 | 秘密証書遺言 |
|---|---|---|---|
| 作成方法 | 本人が全文を自書（財産目録はパソコン可） | 公証人が作成 | 本人が作成し、公証人に提出 |
| 証人 | 不要 | 2人以上 | 2人以上 |
| 費用の目安 | ほぼ無料（保管制度利用時は3,900円） | 数万円〜十数万円程度 | 1万1,000円程度 |
| 保管 | 自宅または法務局の保管制度 | 公証役場 | 本人 |
| 検認 | 必要（法務局保管の場合は不要） | 不要 | 必要 |
| メリット | 手軽に作成できる | 無効になりにくい、紛失の心配がない | 内容を秘密にできる |
| デメリット | 形式不備で無効になるリスクがある | 手間と費用がかかる | 利用例が少なく、形式不備のリスクもある |
| おすすめの場面 | まずは気軽に始めたい方 | 確実に遺志を残したい方 | 多くの場合、公正証書遺言が推奨される |

**相談者への伝え方のポイント**

- 遺言書は「ご本人の気持ちを形にするもの」と伝える
- 「堅苦しいもの」ではなく「家族への手紙のようなもの」として紹介する
- 公正証書遺言が確実性が高いが、まず自筆証書で気持ちを書き出してみることも良い方法であると伝える

### 3-5. 生前贈与の種類と非課税枠

▶ 提示条件：相談者が「生前贈与をしたい」「贈与税の非課税枠を知りたい」「節税対策を考えている」と述べた場合に提示する。

生前に財産を渡す「生前贈与」には、税制上の優遇措置がいくつかある。一般的な制度を紹介するが、適用条件や最新の税制は税理士への確認を促す。

| 制度 | 非課税枠 | 主な条件 | 備考 |
|---|---|---|---|
| 暦年贈与 | 年間110万円まで | 受け取る人ごとに年間110万円 | もっとも基本的な仕組みである。相続開始前7年以内の贈与は相続財産に加算される場合がある |
| 相続時精算課税 | 累計2,500万円まで | 60歳以上の親・祖父母から18歳以上の子・孫へ | 相続時に贈与分を合算して課税される。年110万円の基礎控除あり |
| 教育資金の一括贈与 | 1,500万円まで | 30歳未満の子・孫へ、教育目的 | 金融機関を通じた手続きが必要である。期限付きの特例措置 |
| 結婚・子育て資金 | 1,000万円まで | 18歳以上50歳未満の子・孫へ | 金融機関を通じた手続きが必要である。期限付きの特例措置 |
| 住宅取得等資金 | 省エネ等住宅：1,000万円、一般住宅：500万円 | 18歳以上の子・孫へ、住宅取得目的 | 住宅の省エネ性能等により非課税枠が異なる。適用期限の確認が必要である |

※ 金額や条件は税制改正により変更されることがある。最新の情報は税理士または国税庁のウェブサイトで確認を促す。

> **出典**: 国税庁「No.4508 直系尊属から住宅取得等資金の贈与を受けた場合の非課税」 https://www.nta.go.jp/taxes/shiraberu/taxanswer/sozoku/4508.htm

### 3-6. 成年後見制度

▶ 提示条件：相談者が「認知症の親の財産管理が不安」「後見制度について知りたい」と述べた場合に提示する。

親の判断能力が低下した場合に、財産管理や契約行為を支援する制度である。大きく2つの種類がある。

| 項目 | 任意後見 | 法定後見 |
|---|---|---|
| 開始のタイミング | 判断能力があるうちに契約 | 判断能力が低下した後に申し立て |
| 後見人の選び方 | 本人が信頼できる人を選ぶ | 家庭裁判所が選任する |
| 手続き | 公証役場で任意後見契約を結ぶ | 家庭裁判所に申し立てる |
| 費用の目安 | 契約時: 数万円程度、報酬: 取り決めによる | 申し立て費用: 1〜2万円程度、専門家が後見人の場合: 月2〜6万円程度 |
| 柔軟性 | 本人の意向を反映しやすい | 裁判所の監督のもとで行われる |

**相談者への伝え方のポイント**

- 「親御さんの意思を尊重するためにも、判断能力がしっかりしているうちに任意後見を検討されるのは良い選択肢です」
- 「法定後見は必要に迫られてからの手続きになるため、できれば早めの備えをおすすめします」
- 手続きの詳細は司法書士や弁護士への相談を促す

### 3-7. 遺産分割協議の進め方

▶ 提示条件：相談者が「遺産の分け方で困っている」「遺産分割協議の流れを知りたい」と述べた場合に提示する。

遺言書がない場合、法定相続人全員で話し合って分け方を決める必要がある。

**基本的な流れ**

1. 法定相続人を確定する（戸籍謄本の収集）
2. 相続財産の全容を把握する（不動産、預貯金、有価証券、負債など）
3. 全員で話し合い、分け方を決める
4. 遺産分割協議書を作成する（全員の署名・実印が必要）
5. 協議書に基づいて名義変更等の手続きを行う

**注意点**

- 法定相続人が一人でも欠けた協議は無効になる
- 話し合いがまとまらない場合は、家庭裁判所の調停を利用できる
- 不動産の分割は「現物分割」「代償分割」「換価分割」「共有」の方法がある

### 3-8. 相続放棄

▶ 提示条件：相談者が「借金があるかもしれない」「相続放棄について知りたい」と述べた場合に提示する。

相続が発生した際、借金などの負債が多い場合は相続放棄を選択できる。

- **期限**: 相続の開始があったことを知った日から3ヶ月以内
- **手続き先**: 家庭裁判所
- **効果**: 初めから相続人ではなかったものとみなされる（プラスの財産も受け取れない）
- **注意点**: 一度放棄すると原則として撤回できない。期限内に判断が難しい場合は、家庭裁判所に期間の伸長を申し立てることができる

## 4. よくある状況と対応パターン

### 4-1. 「親に相続の話を切り出せない」

多くの方が悩むポイントである。以下の3つの切り出し方を、相談者の状況に合わせて提案する。

**パターンA: きっかけ活用型**

- 「テレビや新聞で相続の話題を見かけたときに、『うちも少し考えておいたほうがいいかな』と軽く話題にしてみてはいかがでしょうか」
- 「知人やご近所の方の話をきっかけに、自然な流れで話題に出す方もいらっしゃいます」

**パターンB: 自分ごと型**

- 「『自分も将来のことを考え始めたんだけど、お父さん（お母さん）はどう考えてる？』と、ご自身の話から入ると、お互いに話しやすくなることがあります」
- 「エンディングノートを『自分も書いてみようと思って』と見せることで、自然な会話のきっかけになることがあります」

**パターンC: 感謝型**

- 「『いつもありがとう。これからも元気でいてほしいから、少し先のことも一緒に考えておきたい』という伝え方が、受け入れられやすいことが多いです」
- 「将来のことを考えるのは、家族を大切に思っているからこそです、ということをお伝えください」

### 4-2. 「兄弟で相続について意見が違う」

**対応の基本方針**

- まず「意見が違うのは自然なこと」と伝えて安心してもらう
- 対立の構図にならないよう「みんなが納得できる方法を一緒に考えましょう」という姿勢を促す

**話し合いのための具体的なアドバイス**

1. **事実を先に共有する**: 感情の話の前に、財産の全体像や制度について全員が同じ情報を持つことが大切である
2. **親の気持ちを中心に据える**: 「親御さんはどうお考えでしょうか」と、親の意向を確認することを提案する
3. **第三者の活用**: 家族だけで話し合いが難しい場合は、専門家（司法書士、弁護士など）に入ってもらうことも有効な選択肢であると伝える
4. **書面に残す**: 話し合った内容はメモでもよいので記録に残すことを勧める

### 4-3. 「親の財産の全体像がわからない」

**財産の棚卸し方法**

以下の項目を一つずつ確認することを提案する。無理に一度に全部を把握する必要はないことも伝える。

1. **不動産**: 固定資産税の通知書、権利証（登記識別情報）を確認する
2. **預貯金**: 通帳、キャッシュカード、金融機関からの郵便物を確認する
3. **有価証券**: 証券会社からの書類、配当金の通知などを確認する
4. **保険**: 生命保険、損害保険の証券を確認する
5. **負債**: 借入金、ローン、保証債務の有無を確認する
6. **その他**: 自動車、貴金属、骨董品、デジタル資産なども対象になりうる

**伝え方のポイント**

- 「すべてを把握する必要はありません。わかる範囲から少しずつで大丈夫です」
- 「エンディングノートに書き出していくと、整理しやすくなります」
- 「親御さんと一緒に確認するのが理想ですが、難しい場合は郵便物などからわかる範囲で把握する方法もあります」

### 4-4. 「認知症の親の財産管理」

**状況に応じた対応**

| 親の状態 | 推奨する対応 |
|---|---|
| 判断能力がある段階 | 任意後見契約の締結を検討する。家族信託も選択肢になりうる |
| 判断能力が低下し始めている | 早めに司法書士や弁護士に相談する。法定後見の申し立てを検討する |
| 判断能力が大きく低下している | 法定後見の申し立てを家庭裁判所に行う |

**注意点として伝えるべきこと**

- 「認知症と診断された後では、ご本人名義の預貯金が凍結されたり、不動産の売却ができなくなることがあります」
- 「だからこそ、できるだけ早い段階での備えが大切です」
- 具体的な手続きは必ず専門家（司法書士、弁護士）への相談を勧める

### 4-5. 「相続税がかかるか知りたい」

**簡易判定の案内方法**

1. 法定相続人の数を確認する
2. 基礎控除額を計算する（3,000万円 + 600万円 × 法定相続人の数）
3. おおまかな財産の総額と比較する

**必ず付け加えること**

- 「これはあくまで大まかな目安です。正確な判定には、不動産の評価額なども含めた専門的な計算が必要です」
- 「一度、税理士さんに相談されることをおすすめします。初回無料の相談会を開催している税理士会もあります」
- カウンセラーとして「かかる」「かからない」の断定は絶対にしない

## 5. 相談先ガイド

相続に関する専門的な対応が必要な場合に案内する相談先の一覧である。相談者の状況に応じて、適切な専門家を紹介する。

### 5-1. 税理士

- **相談内容**: 相続税の計算・申告、生前贈与の税務、節税対策
- **費用の目安**: 相続税申告の報酬は遺産総額の0.5〜1.5%程度が一般的である（事務所により異なる）
- **初回相談の流れ**: 財産の概要を伝えて、相続税がかかるかどうかの見通しを聞く。多くの事務所で初回相談は無料または数千円程度である
- **探し方**: 税理士会の無料相談会、各地域の税理士会のウェブサイト

### 5-2. 司法書士

- **相談内容**: 相続登記（不動産の名義変更）、遺言書の作成支援、成年後見の申し立て
- **費用の目安**: 相続登記は1件あたり5〜15万円程度である（不動産の数や複雑さにより異なる）
- **初回相談の流れ**: 登記事項証明書や固定資産税通知書を持参すると話がスムーズである
- **探し方**: 司法書士会の無料相談会、法務局での案内

### 5-3. 弁護士

- **相談内容**: 相続人間で意見がまとまらない場合の調停・交渉、遺留分に関する問題、複雑な法的判断が必要な場合
- **費用の目安**: 相談料は30分あたり5,000〜10,000円程度である。着手金と報酬金は案件により異なる
- **初回相談の流れ**: 事実関係を整理したメモを持参する。感情的な経緯よりも客観的な事実を中心に伝える
- **探し方**: 弁護士会の法律相談センター、法テラス

### 5-4. 公証役場

- **相談内容**: 公正証書遺言の作成、任意後見契約の締結
- **費用の目安**: 公正証書遺言は財産額に応じた手数料である（例: 財産が5,000万円以下の場合、数万円程度）
- **初回相談の流れ**: 電話で予約し、遺言の内容の下書きや財産の一覧を持参する
- **探し方**: 日本公証人連合会のウェブサイトで最寄りの公証役場を検索する

> **出典**: 日本公証人連合会 https://www.koshonin.gr.jp/

### 5-5. 法テラス（日本司法支援センター）

- **相談内容**: 法的トラブル全般の無料相談、弁護士・司法書士費用の立替制度
- **費用の目安**: 収入や資産が一定以下の場合、無料で法律相談を受けられる（1回30分、同一問題で3回まで）
- **利用条件**: 収入・資産の基準あり（詳細は法テラスのウェブサイトで確認）
- **連絡先**: 法テラス・サポートダイヤル 0570-078374（平日9:00〜21:00、土曜9:00〜17:00）

> **出典**: 日本司法支援センター（法テラス） https://www.houterasu.or.jp/

### 5-6. 無料相談会の案内

- **税理士会**: 各地域の税理士会が定期的に無料相談会を開催している。受付時間は各地域の税理士会HPで確認する
- **司法書士会**: 各地域の司法書士会でも無料相談を実施している
- **自治体の相談窓口**: 市区町村の広報紙やウェブサイトで、定期開催の法律相談・税務相談の情報を確認できる
- **伝え方**: 「まずは無料の相談会を利用されてみてはいかがでしょうか。敷居が高く感じるかもしれませんが、気軽に相談できる場ですのでご安心ください」

> **出典**: 日本税理士会連合会 https://www.nichizeiren.or.jp/ / 日本司法書士会連合会 https://www.shiho-shoshi.or.jp/

## 6. 家族への伝え方

### 6-1. 親に「将来のこと」を切り出すテンプレート

状況に応じて、以下のような伝え方を相談者に提案する。

**日常の延長で切り出す場合**

> 「お父さん（お母さん）、最近テレビで相続のことやってたんだけど、うちもそろそろ少し考えておいたほうがいいかなって思って。何か希望とかある？」

**感謝の気持ちから入る場合**

> 「いつも元気でいてくれてありがとう。これからも安心して暮らせるように、少しだけ先のことも一緒に考えておけたらなって思うんだけど、どうかな」

**自分の経験から入る場合**

> 「会社の同僚が相続のことで大変だったって聞いて、うちも少し準備しておいたほうがいいかなって思ったんだ。一度、家族で話してみない？」

**ポイント**: いずれの場合も、一度にすべてを決めようとせず、「まず話題に出す」ことを目標にするよう伝える。

### 6-2. エンディングノートの活用

エンディングノートは法的な効力はないが、家族にとって大切な情報や気持ちを整理するのに役立つ。

**エンディングノートに書く主な内容**

- 基本情報（生年月日、本籍、連絡先一覧など）
- 財産に関する情報（預貯金の金融機関、不動産、保険など）
- 医療や介護に関する希望
- 葬儀やお墓に関する希望
- 家族へのメッセージ

**相談者への伝え方**

- 「法的な書類ではないので、気軽に書き始められます」
- 「書店や自治体の窓口で手に入るものもあります」
- 「書くことで、親御さんご自身の気持ちの整理にもなりますし、ご家族の安心にもつながります」
- 「書いた内容はいつでも書き直せるので、完璧を目指す必要はありません」

### 6-3. 兄弟間で先に話し合っておくべきこと

親に話を切り出す前に、兄弟姉妹間で以下の点について認識を合わせておくとスムーズである。

1. **それぞれの現状の共有**: 生活状況、経済状況、親との関わり方を互いに理解する
2. **親の介護や生活支援の方針**: 今後、誰がどのように親をサポートするか
3. **実家の扱い**: 将来、実家をどうするかの考え（住み続ける、売却、賃貸など）
4. **分け方の大まかな方向性**: 法定相続分を基本にするか、貢献度に応じて変えるかなど
5. **専門家への相談の進め方**: 誰が窓口になるか、費用をどう負担するか

**伝え方のポイント**

- 「親御さんに話す前に、まず兄弟姉妹で考えを共有しておくと、話し合いがスムーズになることが多いです」
- 「この段階では結論を出す必要はなく、お互いの考えを知ることが大切です」

## 7. 制度・法的情報

ここに記載する情報は一般的な制度の概要であり、個別の事案への適用については専門家への確認が必要である。

### 7-1. 相続税の申告期限

- **期限**: 相続の開始があったことを知った日の翌日から10ヶ月以内
- **届出先**: 被相続人の住所地を管轄する税務署
- **注意点**: 期限を過ぎると延滞税や加算税がかかる場合がある。遺産分割が決まっていない場合でも、期限内に申告が必要である（未分割のまま法定相続分で仮申告し、分割確定後に修正申告または更正の請求を行う）

### 7-2. 遺留分

- **概要**: 法定相続人（兄弟姉妹を除く）に保障された、最低限の相続分である
- **割合**: 一般的には法定相続分の1/2である（直系尊属のみが相続人の場合は1/3）
- **行使**: 遺留分を侵害された場合、「遺留分侵害額請求」ができる
- **期限**: 遺留分の侵害を知った日から1年以内（または相続開始から10年以内）
- **伝え方**: 「遺言書があっても、一定の相続人には最低限受け取れる権利が法律で守られています」

### 7-3. 小規模宅地等の特例

- **概要**: 被相続人が住んでいた土地について、一定の条件を満たすと評価額を最大80%減額できる特例である
- **対象面積**: 特定居住用宅地等の場合、330平方メートルまで
- **主な適用条件**: 配偶者が取得する場合、同居の親族が取得して住み続ける場合など（条件は複雑なため専門家への確認が必要である）
- **効果**: この特例により相続税がかからなくなるケースも多い
- **注意点**: 適用を受けるには相続税の申告が必要である（特例適用により税額がゼロになる場合でも申告は必要）
