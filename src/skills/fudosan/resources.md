## 5. 相談先ガイド

### 5-1. 自治体の空き家相談窓口

多くの市区町村が空き家に関する相談窓口を設けている。

- **担当部署**: 住宅課、建築指導課、都市計画課など（自治体によって異なる）
- **相談できること**: 空き家の管理に関する助言、補助金制度の案内、空き家バンクへの登録、特定空き家に関する相談
- **費用**: 無料
- **利用方法**: 電話または窓口で予約。実家のある自治体に相談する
- **電話受付時間**: 平日8:30〜17:15が一般的（自治体により異なるため、事前にホームページで確認すること）
- **ポイント**: 自治体によっては、司法書士や不動産鑑定士との無料相談会を定期的に開催している場合がある

### 5-2. 地元の不動産会社

#### 大手と地場の使い分け

| 項目 | 大手不動産会社 | 地場の不動産会社 |
|---|---|---|
| 強み | 広域のネットワーク、知名度による集客力、営業体制が整っている | 地域の事情に詳しい、柔軟な対応、地元の買い手情報を持っている |
| 向いているケース | 都市部やその近郊の物件、比較的新しい物件 | 地方の物件、古い物件、地域特有の事情がある物件 |
| 注意点 | 地方の物件は取り扱いに消極的な場合がある | 会社によってサービスの質に差がある |

- 迷う場合は、大手1社と地場1〜2社の両方に査定を依頼して比較するとよい
- 媒介契約（専任・一般など）の違いも確認しておくと安心

### 5-3. 空き家バンク

空き家バンクは、自治体やその委託先が運営する空き家の情報提供サービスである。

- **仕組み**: 空き家の所有者が物件を登録し、利用希望者とマッチングする
- **費用**: 登録は基本的に無料
- **メリット**: 通常の不動産市場では流通しにくい物件でも、移住希望者や地域活性化に関心のある人に届く可能性がある
- **注意点**: マッチングまでに時間がかかることがある。契約手続きには不動産会社を介することが推奨される
- **確認方法**: 実家のある自治体のホームページで「空き家バンク」と検索する。全国版として「全国版空き家・空き地バンク」（国土交通省推進）もある

### 5-4. 司法書士

相続登記の手続きを依頼する場合の主な相談先である。

- **相談できること**: 相続登記の手続き、遺産分割協議書の作成支援、不動産の名義変更、成年後見や家族信託に関する相談
- **費用の目安**: 相続登記の報酬は5万〜15万円程度（物件数や相続人の数により変動）。別途、登録免許税（固定資産税評価額の0.4%）がかかる
- **探し方**: 各地の司法書士会のホームページで検索できる。法務局の窓口でも紹介を受けられる場合がある
- **ポイント**: 初回相談が無料の事務所も多い。複数の事務所に相談して比較することも可能

### 5-5. その他の相談先

| 相談先 | 相談できること | 費用 |
|---|---|---|
| 税理士 | 譲渡所得税、相続税、固定資産税に関する相談 | 初回無料〜数千円（事務所による） |
| 行政書士 | 遺産分割協議書の作成、各種届出の代行 | 数万円〜（内容による） |
| 弁護士 | 相続トラブル、兄弟間の紛争、境界問題 | 初回相談30分 5,000〜10,000円程度（事務所による） |
| 土地家屋調査士 | 土地の境界確定、測量 | 30万〜80万円程度（面積・条件による） |
| 法テラス（日本司法支援センター） | 経済的に余裕がない場合の法律相談 | 無料（収入要件あり）。電話受付: 0570-078374（平日9:00〜21:00、土曜9:00〜17:00） |

## 6. 家族への伝え方

### 6-1. 親に「実家をどうするか」を切り出すタイミングと言い方

#### 適切なタイミング

- 親の体調や気持ちが安定しているとき
- 帰省時など、自然に家の話題が出やすい場面
- 近所で空き家に関するニュースや話題があったとき
- 親自身が「この家も古くなったね」「いつまで住めるかな」などの発言をしたとき

#### 避けるべきタイミング

- 親の体調が優れないとき
- 家族の間でほかの問題を抱えているとき
- 慌ただしい場面や時間がないとき

#### 切り出し方の例

- 「最近、テレビで空き家の特集をやっていたんだけど、うちの家のことも少し考えておいた方がいいかなって思って」
- 「お父さん（お母さん）が元気なうちに、家のことについてお父さん（お母さん）の考えを聞いておきたいなと思って」
- 「近所の○○さんの家が空き家になっていたけど、うちも将来のことを少し話しておけたらと思うんだけど」

#### 伝え方のポイント

- 「決めよう」ではなく「考えを聞かせてほしい」という姿勢で
- 一度の会話ですべてを決めようとしない
- 親の気持ちや考えを最優先で聞く
- 「あなたのことが心配だから」という気持ちを素直に伝える

### 6-2. 兄弟間で合意形成するステップ

1. **情報収集**: 実家の現状（築年数、固定資産税額、維持費、周辺相場）を客観的に整理し、全員で共有する
2. **各自の気持ちを共有**: それぞれの思いや事情を、否定せずに聞き合う場を設ける。対面が難しければオンライン会議でもよい
3. **選択肢の比較**: 売却・賃貸・管理維持のメリット・デメリット・費用を一覧にして比較する
4. **役割分担の確認**: 誰が何を担当するか、費用は誰がどう負担するかを明確にする
5. **方向性の合意**: 全員が100%満足する答えは難しいことを前提に、全員が「納得できる」方向性を探す
6. **期限の設定**: 「いつまでに次のステップに進むか」をゆるやかに決める
7. **記録に残す**: 話し合いの内容は簡単でもよいのでメモに残しておく。後々の認識の違いを防ぐため

## 7. 制度・法的情報

### 7-1. 相続登記義務化の詳細

| 項目 | 内容 |
|---|---|
| 施行日 | 2024年4月1日 |
| 対象 | 相続（遺言を含む）により不動産を取得したすべてのケース |
| 申請期限 | 相続により不動産を取得したことを知った日から3年以内 |
| 経過措置 | 施行日前に発生した相続も対象。施行日から3年以内（2027年3月31日まで）に申請が必要 |
| 罰則 | 正当な理由なく期限内に申請を怠った場合、10万円以下の過料 |
| 正当な理由の例 | 相続人が多数で調査に時間を要する場合、遺言の有効性について争いがある場合など |
| 簡便な制度 | 「相続人申告登記」制度を利用すれば、遺産分割が完了する前でも、申告義務を履行できる |

**出典**: 法務省「不動産の相続登記が義務化されます」
**根拠法**: 不動産登記法（令和3年法律第24号による改正）

### 7-2. 空き家の譲渡所得3,000万円特別控除

相続した空き家を売却した場合に、一定の条件を満たせば譲渡所得から最大3,000万円を控除できる制度である。

| 項目 | 内容 |
|---|---|
| 正式名称 | 被相続人の居住用財産（空き家）に係る譲渡所得の特別控除の特例 |
| 控除額 | 最大3,000万円（相続人が3人以上の場合は2,000万円。2024年1月以降の譲渡が対象） |
| 適用期限 | 2027年12月31日までの譲渡（延長の可能性あり。最新情報を確認すること） |
| 主な適用条件 | (1) 被相続人が一人で居住していた家屋であること (2) 1981年（昭和56年）5月31日以前に建築された家屋であること（旧耐震基準） (3) 相続から譲渡までの間に、事業・貸付・居住の用に供されていないこと (4) 相続の開始があった日から3年を経過する日の属する年の12月31日までに譲渡すること (5) 譲渡価額が1億円以下であること (6) 耐震リフォームを行うか、建物を解体して更地で売却すること |
| 申請先 | 確定申告時に税務署に申請する |

**注意**: 条件が複雑なため、適用の可否は税理士や税務署に確認することを推奨する。

**出典**: 国税庁「被相続人の居住用財産（空き家）を売ったときの特例」（No.3306）
**根拠法**: 租税特別措置法第35条第3項

### 7-3. 各自治体の空き家対策補助金

自治体ごとに様々な補助金・助成金制度がある。代表的な種類は以下のとおりである。

| 補助金の種類 | 概要 | 金額の目安 |
|---|---|---|
| 解体費補助 | 老朽化した空き家の解体費用を補助 | 上限50万〜100万円程度 |
| 改修費補助 | 空き家のリフォーム・改修費用を補助（移住者向けなど） | 上限50万〜200万円程度 |
| 空き家バンク登録奨励金 | 空き家バンクに登録した所有者への奨励金 | 5万〜20万円程度 |
| 家財道具撤去費補助 | 空き家内の残置物の撤去費用を補助 | 上限10万〜30万円程度 |

- 制度の有無、金額、条件は自治体によって大きく異なる
- 年度ごとに予算枠があるため、早めの確認が望ましい
- 実家のある自治体のホームページまたは窓口で最新情報を確認すること

**確認方法**: 「（自治体名） 空き家 補助金」で検索するか、自治体の住宅関連部署に問い合わせる
