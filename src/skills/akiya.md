# 実家と空き家に関する相談スキル

> 対象: 高齢の親を心配する40〜60代の家族
> テーマ: 実家の管理・売却・賃貸、相続登記、空き家対策法、固定資産税
> 最終更新: 2026年3月

---

## 1. このスキルの対象となる相談例

以下のような相談が寄せられた場合に、このスキルの知識を活用して回答する。

1. 「実家が空き家になりそうで、どうしたらいいかわからない」
2. 「親が施設に入ったあと、家をどうすればいいか悩んでいる」
3. 「実家を相続したが、遠方に住んでいて管理できない」
4. 「兄弟で実家の扱いについて意見が合わない」
5. 「古い実家に思い出があって手放せないが、管理も難しい」
6. 「空き家にかかる税金や維持費が心配」
7. 「相続登記が義務化されたと聞いたが、何をすればいいかわからない」
8. 「実家を売りたいが、古すぎて買い手がつくか不安」

## 2. カウンセリングの心得

### 最も大切な姿勢

実家は単なる不動産ではない。相談者にとって、幼少期の思い出、家族の歴史、親との絆が詰まった場所である。まず感情面を十分に受け止めてから、実務的な話題に入ること。

### 絶対に使ってはいけない言葉

以下の表現は、相談者の気持ちを傷つけたり、焦りを生む可能性がある。

| 使ってはいけない表現 | 問題点 |
|---|---|
| 「早く売った方がいいですよ」 | 焦りを煽り、感情の整理を妨げる |
| 「放置するとリスクがあります」 | 不安を増幅させ、脅しのように聞こえる |
| 「資産価値がないですね」 | 思い出の場所を否定することになる |
| 「もったいない」 | 相談者の判断を否定する印象を与える |
| 「普通はこうします」 | 家族ごとに事情が異なることを無視している |
| 「損をしますよ」 | 金銭的な合理性だけで判断を迫る印象を与える |

### 使うべき言い換え表現

- 「大切なお家のことですから、焦らず一つずつ整理していきましょう」
- 「ご家族の思い出が詰まった場所ですよね。お気持ちは自然なことです」
- 「すぐに決めなくても大丈夫です。まずは選択肢を知るところから始めましょう」
- 「いろいろな方法がありますので、ご家族に合ったやり方を一緒に考えましょう」
- 「お気持ちの整理と、実務的な準備は、別々に進めて構いません」

### 揺れる気持ちへの寄り添い

「合理的に判断すべき」という頭の声と、「思い出を大切にしたい」という心の声の間で揺れることは、ごく自然なことである。カウンセラーは以下の点を意識する。

- どちらの気持ちも否定しない。「迷って当然です」と伝える
- 合理的な判断を急がせない。感情の整理には時間がかかることを前提とする
- 「思い出を残す方法」と「家を手放す判断」は両立できることを伝える（写真を撮る、一部の建材を残す、など）
- 相談者自身が納得できるタイミングを尊重する

### 兄弟間の意見の相違について

実家の扱いは、兄弟間で意見が割れることが非常に多い分野である。以下を理解しておく。

- 実家との距離（物理的・心理的）が兄弟間で異なるため、温度差が生じやすい
- 相続の公平性に関する考え方も兄弟で異なることが多い
- 「誰が管理するのか」「費用は誰が負担するのか」が争点になりやすい
- カウンセラーは特定の立場に肩入れせず、全員の気持ちを尊重する姿勢を保つ
- 「正解は一つではない」ことを前提に、話し合いの進め方を提案する

## 3. 基礎知識

### 3-1. 空き家の3つの選択肢

#### 売却

| 項目 | 内容 |
|---|---|
| メリット | 維持費・固定資産税の負担がなくなる。まとまった資金が得られる。管理の手間から解放される |
| デメリット | 思い出の場所を手放すことになる。希望価格で売れるとは限らない。売却までに時間がかかることがある |
| 費用の目安 | 仲介手数料（売却価格の3%+6万円+消費税）、測量費（30万〜80万円程度）、残置物の撤去費（20万〜50万円程度） |
| 手間 | 不動産会社の選定、内覧対応、契約手続き、残置物の整理 |
| 税金への影響 | 譲渡所得税が発生する場合がある。ただし一定の条件を満たせば3,000万円の特別控除が使える（詳細は「7. 制度・法的情報」を参照） |

#### 賃貸

| 項目 | 内容 |
|---|---|
| メリット | 家を手放さずに収入が得られる。建物の劣化を抑えられる（人が住むことで換気・通水が行われる） |
| デメリット | 入居者の募集・管理の手間がかかる。修繕費用が発生する。借主とのトラブルリスクがある。古い家はそのままでは借り手がつきにくい |
| 費用の目安 | リフォーム費（100万〜500万円程度、規模による）、管理委託費（家賃の5〜10%程度）、修繕積立 |
| 手間 | リフォームの検討、管理会社の選定、入居者対応（管理委託しない場合） |
| 税金への影響 | 家賃収入に対して所得税・住民税が発生する。経費として修繕費や管理費を計上できる |

#### 管理維持（当面そのまま保有する）

| 項目 | 内容 |
|---|---|
| メリット | すぐに判断しなくてよい。家族で話し合う時間を確保できる。将来の選択肢を残せる |
| デメリット | 維持費・固定資産税が継続的にかかる。建物が劣化する。近隣への影響が生じる可能性がある |
| 費用の目安 | 空き家管理サービス（月額5,000円〜15,000円程度）、固定資産税（物件による）、火災保険、最低限の修繕費 |
| 手間 | 定期的な換気・通水・草刈り・郵便物の確認。遠方の場合は管理サービスへの委託 |
| 税金への影響 | 住宅用地特例が適用されている間は、固定資産税が軽減される。ただし特定空き家に指定されると特例が解除される可能性がある |

### 3-2. 空き家対策特別措置法

空き家対策特別措置法（空家等対策の推進に関する特別措置法）は、増加する空き家問題に対応するために制定された法律である。

#### 特定空き家に指定されるとどうなるか

自治体が以下のいずれかに該当すると判断した空き家は「特定空き家」に指定される可能性がある。

- 倒壊など著しく保安上危険となるおそれがある
- 著しく衛生上有害となるおそれがある
- 適切な管理が行われていないことにより著しく景観を損なっている
- その他周辺の生活環境の保全を図るために放置することが不適切である

特定空き家に指定されると、以下の措置が段階的に行われる。

1. **助言・指導**: 自治体から改善を求められる
2. **勧告**: 改善されない場合、固定資産税の住宅用地特例が解除される（税額が最大で約6倍になる可能性がある）
3. **命令**: 勧告に従わない場合、50万円以下の過料が科される可能性がある
4. **行政代執行**: 命令にも従わない場合、自治体が強制的に解体などを行い、費用を所有者に請求する

#### 管理不全空き家（2023年の法改正で追加）

2023年の法改正により、特定空き家の一歩手前の状態にある「管理不全空き家」という区分が追加された。管理不全空き家に対しても、自治体は指導・勧告を行うことができ、勧告を受けると固定資産税の住宅用地特例が解除される。

つまり、特定空き家に至る前の段階でも、適切な管理を怠ると税負担が増える可能性がある。

### 3-3. 相続登記の義務化

2024年4月1日から、不動産の相続登記が義務化された。

| 項目 | 内容 |
|---|---|
| 対象 | 相続により不動産を取得した人 |
| 期限 | 相続を知った日から3年以内 |
| 罰則 | 正当な理由なく申請を怠った場合、10万円以下の過料 |
| 過去の相続 | 2024年4月1日より前に発生した相続も対象（施行日から3年以内、つまり2027年3月末までに登記が必要） |
| 申請先 | 不動産の所在地を管轄する法務局 |

相続登記を行わないまま放置すると、世代を重ねるごとに相続人が増え、手続きが複雑になる。早めの対応が望ましいが、相談者を焦らせる伝え方は避けること。

### 3-4. 固定資産税の住宅用地特例

住宅が建っている土地には「住宅用地特例」が適用され、固定資産税が軽減されている。

| 区分 | 軽減率 |
|---|---|
| 小規模住宅用地（200平方メートル以下の部分） | 固定資産税が6分の1に軽減 |
| 一般住宅用地（200平方メートルを超える部分） | 固定資産税が3分の1に軽減 |

**更地にすると税額が上がる問題**: 建物を解体して更地にすると、この特例が適用されなくなるため、固定資産税が最大で約6倍に上がる可能性がある。このため「解体した方がよいのか、建物を残した方がよいのか」は慎重に検討する必要がある。

ただし、特定空き家や管理不全空き家に指定されて勧告を受けた場合も特例が解除されるため、「建物を残しておけば安心」とは一概に言えない点に注意する。

### 3-5. 空き家の管理方法と費用相場

自分で管理する場合と、管理サービスに委託する場合がある。

#### 自分で管理する場合のチェックリスト

- 月1回程度の換気（窓を開けて空気を入れ替える）
- 通水（水道を数分間流して、排水トラップの封水を維持する）
- 郵便物の回収
- 庭の草刈り・植木の手入れ（季節による）
- 外観の点検（屋根、外壁、雨樋など）
- 近隣への挨拶と連絡先の共有

#### 管理サービスに委託する場合の費用相場

| サービス内容 | 月額の目安 |
|---|---|
| 基本プラン（月1回の巡回・換気・通水・外観点検・報告） | 5,000円〜10,000円程度 |
| 充実プラン（上記に加え、庭の手入れ・室内清掃・郵便物転送） | 10,000円〜15,000円程度 |
| スポット対応（台風後の点検、緊急対応など） | 1回あたり5,000円〜15,000円程度 |

管理サービスは、不動産会社、シルバー人材センター、空き家管理専門の会社などが提供している。

### 3-6. 解体費用の相場と補助金制度

#### 解体費用の相場

解体費用は建物の構造、面積、立地条件、地域によって大きく異なる。

| 建物の構造 | 坪単価の目安 |
|---|---|
| 木造 | 3万〜5万円程度 |
| 鉄骨造 | 4万〜7万円程度 |
| 鉄筋コンクリート造（RC造） | 5万〜8万円程度 |

たとえば木造30坪の家であれば、90万〜150万円程度が目安となる。ただし、アスベスト含有材の処理、浄化槽の撤去、重機が入りにくい狭い立地などの条件により、費用が大きく上振れすることがある。

#### 自治体の補助金制度

多くの自治体が空き家の解体に対する補助金制度を設けている。

- 補助額の目安: 解体費用の2分の1〜5分の3程度、上限50万〜100万円程度が多い
- 対象条件: 自治体によって異なるが、「旧耐震基準の建物」「特定空き家に該当するもの」「一定期間使用されていないもの」などの条件がつくことが多い
- 申請時期: 年度初めに受付開始し、予算に達し次第終了するケースが多い
- 確認方法: お住まいの自治体、または実家のある自治体のホームページや窓口で確認する

## 4. よくある状況と対応パターン

### 4-1. 「親が存命で名義は親のまま」

親が存命の間は、不動産の名義は親のものである。この段階でできることを整理する。

**今できること**

- 親の意向を確認する（家をどうしたいか、誰に託したいか）
- 家族間で将来のことについて話し合いの場を設ける
- 不動産の現状を把握する（築年数、構造、面積、土地の境界、権利関係）
- 固定資産税の納税通知書を確認し、年間の維持費を把握する
- 登記簿謄本（登記事項証明書）を取得して、名義や抵当権の有無を確認する
- 親が認知症になった場合に備えて、任意後見制度や家族信託について情報を集めておく

**伝え方の例**

「今すぐ何かを決める必要はありませんが、お家の基本的な情報を整理しておくと、将来どんな選択をするにしても安心です。お父様（お母様）のお気持ちを聞いてみることも、大切な一歩になります。」

### 4-2. 「兄弟で実家の扱いについて意見が割れている」

**話し合いの進め方**

1. **事実の共有から始める**: 感情や意見の前に、実家の現状（築年数、維持費、固定資産税額、周辺の不動産相場）を全員で共有する
2. **それぞれの気持ちを聞く場をつくる**: 「売りたい」「残したい」の背景にある気持ちを、否定せずに聞き合う
3. **選択肢を並べて比較する**: 売却・賃貸・管理維持のそれぞれについて、費用と手間を具体的に示す
4. **期限を区切る**: 「今すぐ決めなくてよいが、いつまでに方向性を決めるか」を合意する
5. **第三者の活用を提案する**: 話し合いが難航する場合、司法書士や行政書士などの専門家を交えることも選択肢の一つ

**伝え方の例**

「ご兄弟で意見が異なるのは、それぞれにお家への思いや生活の事情があるからこそです。まずはお互いの考えを聞き合うところから始めてみてはいかがでしょうか。事実を一緒に確認するだけでも、話し合いが進みやすくなることがあります。」

### 4-3. 「遠方で管理できない」

**管理サービスの活用**

- 空き家管理サービスを利用すれば、月1回程度の巡回・換気・通水・報告を委託できる（費用は3-5を参照）
- 実家のある地域の不動産会社に相談すると、管理サービスを提供している場合がある
- シルバー人材センターに庭の手入れなどを依頼できる地域もある
- 近隣の方に見守りをお願いできる場合は、連絡先を共有しておくとよい

**伝え方の例**

「遠方にお住まいだと、なかなかこまめに通うのは難しいですよね。最近は空き家の管理を代行してくれるサービスもありますので、ご負担を減らす方法を一緒に考えましょう。」

### 4-4. 「古すぎて売れるかわからない」

**不動産会社への相談の仕方**

- まずは複数の不動産会社（2〜3社程度）に査定を依頼して、相場を把握する
- 築年数が古い場合でも、土地としての価値がある場合は買い手が見つかることがある
- 「古家付き土地」として売り出す方法もある（解体費用を買主負担にする代わりに価格を下げる）
- 不動産会社に相談する際は、登記簿謄本、固定資産税の納税通知書、建物の図面などを持参すると話がスムーズに進む
- 地域によっては空き家バンクへの登録も有効（詳細は5-3を参照）

**伝え方の例**

「築年数が経っているから売れないと決まったわけではありません。まずは地元の不動産会社さんに相談してみると、思っていたよりも選択肢が広がることがあります。無料で査定してくれるところも多いですよ。」

### 4-5. 「思い出があって手放せない」

**感情の整理と時間軸の考え方**

- この気持ちは自然なものであり、否定する必要はまったくない
- 「手放す＝思い出がなくなる」ではないことを伝える
- 思い出を形に残す方法を提案する（家の写真を撮る、家族で最後に集まる、建材の一部を記念に残す、など）
- 「今すぐ決めなくてよい」ことを伝えつつ、維持にかかる費用や手間を客観的に共有する
- 時間軸を意識する（1年後、3年後、5年後にどうなっているかを想像してみる）

**伝え方の例**

「大切な思い出がたくさん詰まったお家ですから、簡単に決められないのは当然のことです。お気持ちの整理には時間がかかりますし、それでいいんです。思い出を残す方法はいろいろありますので、お家をどうするかとは分けて考えることもできますよ。」

## 5. 相談先ガイド

### 5-1. 自治体の空き家相談窓口

多くの市区町村が空き家に関する相談窓口を設けている。

- **担当部署**: 住宅課、建築指導課、都市計画課など（自治体によって異なる）
- **相談できること**: 空き家の管理に関する助言、補助金制度の案内、空き家バンクへの登録、特定空き家に関する相談
- **費用**: 無料
- **利用方法**: 電話または窓口で予約。実家のある自治体に相談する
- **ポイント**: 自治体によっては、司法書士や不動産鑑定士との無料相談会を定期的に開催している場合がある

### 5-2. 地元の不動産会社

#### 大手と地場の使い分け

| 項目 | 大手不動産会社 | 地場の不動産会社 |
|---|---|---|
| 強み | 広域のネットワーク、知名度による集客力、営業体制が整っている | 地域の事情に詳しい、柔軟な対応、地元の買い手情報を持っている |
| 向いているケース | 都市部やその近郊の物件、比較的新しい物件 | 地方の物件、古い物件、地域特有の事情がある物件 |
| 注意点 | 地方の物件は取り扱いに消極的な場合がある | 会社によってサービスの質に差がある |

- 迷う場合は、大手1社と地場1〜2社の両方に査定を依頼して比較するとよい
- 媒介契約（専任・一般など）の違いも確認しておくと安心

### 5-3. 空き家バンク

空き家バンクは、自治体やその委託先が運営する空き家の情報提供サービスである。

- **仕組み**: 空き家の所有者が物件を登録し、利用希望者とマッチングする
- **費用**: 登録は基本的に無料
- **メリット**: 通常の不動産市場では流通しにくい物件でも、移住希望者や地域活性化に関心のある人に届く可能性がある
- **注意点**: マッチングまでに時間がかかることがある。契約手続きには不動産会社を介することが推奨される
- **確認方法**: 実家のある自治体のホームページで「空き家バンク」と検索する。全国版として「全国版空き家・空き地バンク」（国土交通省推進）もある

### 5-4. 司法書士

相続登記の手続きを依頼する場合の主な相談先である。

- **相談できること**: 相続登記の手続き、遺産分割協議書の作成支援、不動産の名義変更、成年後見や家族信託に関する相談
- **費用の目安**: 相続登記の報酬は5万〜15万円程度（物件数や相続人の数により変動）。別途、登録免許税（固定資産税評価額の0.4%）がかかる
- **探し方**: 各地の司法書士会のホームページで検索できる。法務局の窓口でも紹介を受けられる場合がある
- **ポイント**: 初回相談が無料の事務所も多い。複数の事務所に相談して比較することも可能

### 5-5. その他の相談先

| 相談先 | 相談できること | 費用 |
|---|---|---|
| 税理士 | 譲渡所得税、相続税、固定資産税に関する相談 | 初回無料〜数千円（事務所による） |
| 行政書士 | 遺産分割協議書の作成、各種届出の代行 | 数万円〜（内容による） |
| 弁護士 | 相続トラブル、兄弟間の紛争、境界問題 | 初回相談30分5,000円程度〜（事務所による） |
| 土地家屋調査士 | 土地の境界確定、測量 | 30万〜80万円程度（面積・条件による） |
| 法テラス（日本司法支援センター） | 経済的に余裕がない場合の法律相談 | 無料（収入要件あり） |

## 6. 家族への伝え方

### 6-1. 親に「実家をどうするか」を切り出すタイミングと言い方

#### 適切なタイミング

- 親の体調や気持ちが安定しているとき
- 帰省時など、自然に家の話題が出やすい場面
- 近所で空き家に関するニュースや話題があったとき
- 親自身が「この家も古くなったね」「いつまで住めるかな」などの発言をしたとき

#### 避けるべきタイミング

- 親の体調が優れないとき
- 家族の間でほかの問題を抱えているとき
- 慌ただしい場面や時間がないとき

#### 切り出し方の例

- 「最近、テレビで空き家の特集をやっていたんだけど、うちの家のことも少し考えておいた方がいいかなって思って」
- 「お父さん（お母さん）が元気なうちに、家のことについてお父さん（お母さん）の考えを聞いておきたいなと思って」
- 「近所の○○さんの家が空き家になっていたけど、うちも将来のことを少し話しておけたらと思うんだけど」

#### 伝え方のポイント

- 「決めよう」ではなく「考えを聞かせてほしい」という姿勢で
- 一度の会話ですべてを決めようとしない
- 親の気持ちや考えを最優先で聞く
- 「あなたのことが心配だから」という気持ちを素直に伝える

### 6-2. 兄弟間で合意形成するステップ

1. **情報収集**: 実家の現状（築年数、固定資産税額、維持費、周辺相場）を客観的に整理し、全員で共有する
2. **各自の気持ちを共有**: それぞれの思いや事情を、否定せずに聞き合う場を設ける。対面が難しければオンライン会議でもよい
3. **選択肢の比較**: 売却・賃貸・管理維持のメリット・デメリット・費用を一覧にして比較する
4. **役割分担の確認**: 誰が何を担当するか、費用は誰がどう負担するかを明確にする
5. **方向性の合意**: 全員が100%満足する答えは難しいことを前提に、全員が「納得できる」方向性を探す
6. **期限の設定**: 「いつまでに次のステップに進むか」をゆるやかに決める
7. **記録に残す**: 話し合いの内容は簡単でもよいのでメモに残しておく。後々の認識の違いを防ぐため

## 7. 制度・法的情報

### 7-1. 相続登記義務化の詳細

| 項目 | 内容 |
|---|---|
| 施行日 | 2024年4月1日 |
| 対象 | 相続（遺言を含む）により不動産を取得したすべてのケース |
| 申請期限 | 相続により不動産を取得したことを知った日から3年以内 |
| 経過措置 | 施行日前に発生した相続も対象。施行日から3年以内（2027年3月31日まで）に申請が必要 |
| 罰則 | 正当な理由なく期限内に申請を怠った場合、10万円以下の過料 |
| 正当な理由の例 | 相続人が多数で調査に時間を要する場合、遺言の有効性について争いがある場合など |
| 簡便な制度 | 「相続人申告登記」制度を利用すれば、遺産分割が完了する前でも、申告義務を履行できる |

**出典**: 法務省「不動産の相続登記が義務化されます」
**根拠法**: 不動産登記法（令和3年法律第24号による改正）

### 7-2. 空き家の譲渡所得3,000万円特別控除

相続した空き家を売却した場合に、一定の条件を満たせば譲渡所得から最大3,000万円を控除できる制度。

| 項目 | 内容 |
|---|---|
| 正式名称 | 被相続人の居住用財産（空き家）に係る譲渡所得の特別控除の特例 |
| 控除額 | 最大3,000万円（相続人が3人以上の場合は2,000万円） |
| 適用期限 | 2027年12月31日までの譲渡（延長の可能性あり。最新情報を確認すること） |
| 主な適用条件 | (1) 被相続人が一人で居住していた家屋であること (2) 1981年（昭和56年）5月31日以前に建築された家屋であること（旧耐震基準） (3) 相続から譲渡までの間に、事業・貸付・居住の用に供されていないこと (4) 相続の開始があった日から3年を経過する日の属する年の12月31日までに譲渡すること (5) 譲渡価額が1億円以下であること (6) 耐震リフォームを行うか、建物を解体して更地で売却すること |
| 申請先 | 確定申告時に税務署に申請する |

**注意**: 条件が複雑なため、適用の可否は税理士や税務署に確認することを推奨する。

**出典**: 国税庁「被相続人の居住用財産（空き家）を売ったときの特例」
**根拠法**: 租税特別措置法第35条第3項

### 7-3. 各自治体の空き家対策補助金

自治体ごとに様々な補助金・助成金制度がある。代表的な種類は以下のとおり。

| 補助金の種類 | 概要 | 金額の目安 |
|---|---|---|
| 解体費補助 | 老朽化した空き家の解体費用を補助 | 上限50万〜100万円程度 |
| 改修費補助 | 空き家のリフォーム・改修費用を補助（移住者向けなど） | 上限50万〜200万円程度 |
| 空き家バンク登録奨励金 | 空き家バンクに登録した所有者への奨励金 | 5万〜20万円程度 |
| 家財道具撤去費補助 | 空き家内の残置物の撤去費用を補助 | 上限10万〜30万円程度 |

- 制度の有無、金額、条件は自治体によって大きく異なる
- 年度ごとに予算枠があるため、早めの確認が望ましい
- 実家のある自治体のホームページまたは窓口で最新情報を確認すること

**確認方法**: 「（自治体名） 空き家 補助金」で検索するか、自治体の住宅関連部署に問い合わせる
