## 3. 基礎知識

### 3-1. 相続の全体フロー

▶ 提示条件：相談者が「何から始めればいいかわからない」「相続の流れを知りたい」と述べた場合に提示する。

相続に関わる流れは大きく3つの段階に分かれる。

**第1段階: 生前の備え（今からできること）**

1. ご家族で将来のことについて話し合う
2. 財産の全体像を把握する
3. 遺言書の作成を検討する
4. 生前贈与を検討する
5. 必要に応じて任意後見契約を検討する
6. エンディングノートを活用する

**第2段階: 相続の発生**

1. 死亡届の提出（7日以内）
2. 遺言書の有無を確認する
3. 法定相続人を確定する（戸籍の収集）
4. 相続財産の調査を行う
5. 相続放棄の判断（3ヶ月以内）
6. 準確定申告（4ヶ月以内）

**第3段階: 手続きの完了**

1. 遺産分割協議を行う（遺言書がない場合）
2. 遺産分割協議書を作成する
3. 不動産の相続登記を行う（義務化済み・取得を知った日から3年以内）
4. 相続税の申告・納付（10ヶ月以内）
5. 各種名義変更手続きを行う

### 3-2. 法定相続人と法定相続分

▶ 提示条件：相談者が「誰が相続人になるのか」「どのくらいの割合で分けるのか」と質問した場合に提示する。

法律で定められた相続の順位と割合は以下の通りである。一般的な目安として案内し、具体的な判断は専門家への確認を促す。

**相続の順位**

| 順位 | 相続人 | 備考 |
|---|---|---|
| 常に相続人 | 配偶者 | 婚姻関係にある夫または妻 |
| 第1順位 | 子 | 子が先に亡くなっている場合は孫（代襲相続） |
| 第2順位 | 父母 | 第1順位の相続人がいない場合 |
| 第3順位 | 兄弟姉妹 | 第1・第2順位の相続人がいない場合 |

**法定相続分の目安**

| パターン | 配偶者 | 子 | 父母 | 兄弟姉妹 |
|---|---|---|---|---|
| 配偶者と子 | 1/2 | 1/2（人数で均等割り） | - | - |
| 配偶者と父母 | 2/3 | - | 1/3（人数で均等割り） | - |
| 配偶者と兄弟姉妹 | 3/4 | - | - | 1/4（人数で均等割り） |
| 配偶者のみ | すべて | - | - | - |
| 子のみ | - | すべて（均等割り） | - | - |

※ 上記はあくまで法律上の目安であり、遺言書がある場合や、ご家族の話し合いにより異なる分け方にすることも可能である。

### 3-3. 基礎控除と相続税の判断目安

▶ 提示条件：相談者が「相続税がかかるか知りたい」「うちは関係あるのか」と質問した場合に提示する。

相続税には基礎控除があり、財産の総額がこの金額を超えなければ、一般的には相続税の申告は不要とされている。

**基礎控除の計算式**

```
基礎控除額 = 3,000万円 + 600万円 × 法定相続人の数
```

**計算例**

| 法定相続人の数 | 基礎控除額 |
|---|---|
| 1人 | 3,600万円 |
| 2人 | 4,200万円 |
| 3人 | 4,800万円 |
| 4人 | 5,400万円 |

**「うちは関係ある？」と聞かれた場合の対応**

- 「ご自宅の土地・建物も含めて考える必要があります。都市部に不動産をお持ちの場合は、思っていたより評価額が高いこともあります」
- 「まずは、おおまかな財産の全体像を把握されることをおすすめします。その上で、税理士さんに一度ご相談されると安心です」
- 断定的な判断（「かかりません」「かかります」）は絶対にしない。

### 3-4. 遺言書の3種類

▶ 提示条件：相談者が「遺言書を作りたい」「遺言書の種類を知りたい」「親に遺言書を書いてほしい」と述べた場合に提示する。

| 項目 | 自筆証書遺言 | 公正証書遺言 | 秘密証書遺言 |
|---|---|---|---|
| 作成方法 | 本人が全文を自書（財産目録はパソコン可） | 公証人が作成 | 本人が作成し、公証人に提出 |
| 証人 | 不要 | 2人以上 | 2人以上 |
| 費用の目安 | ほぼ無料（保管制度利用時は3,900円） | 数万円〜十数万円程度 | 1万1,000円程度 |
| 保管 | 自宅または法務局の保管制度 | 公証役場 | 本人 |
| 検認 | 必要（法務局保管の場合は不要） | 不要 | 必要 |
| メリット | 手軽に作成できる | 無効になりにくい、紛失の心配がない | 内容を秘密にできる |
| デメリット | 形式不備で無効になるリスクがある | 手間と費用がかかる | 利用例が少なく、形式不備のリスクもある |
| おすすめの場面 | まずは気軽に始めたい方 | 確実に遺志を残したい方 | 多くの場合、公正証書遺言が推奨される |

**相談者への伝え方のポイント**

- 遺言書は「ご本人の気持ちを形にするもの」と伝える
- 「堅苦しいもの」ではなく「家族への手紙のようなもの」として紹介する
- 公正証書遺言が確実性が高いが、まず自筆証書で気持ちを書き出してみることも良い方法であると伝える

### 3-5. 生前贈与の種類と非課税枠

▶ 提示条件：相談者が「生前贈与をしたい」「贈与税の非課税枠を知りたい」「節税対策を考えている」と述べた場合に提示する。

生前に財産を渡す「生前贈与」には、税制上の優遇措置がいくつかある。一般的な制度を紹介するが、適用条件や最新の税制は税理士への確認を促す。

| 制度 | 非課税枠 | 主な条件 | 備考 |
|---|---|---|---|
| 暦年贈与 | 年間110万円まで | 受け取る人ごとに年間110万円 | もっとも基本的な仕組みである。相続開始前7年以内の贈与は相続財産に加算される場合がある |
| 相続時精算課税 | 累計2,500万円まで | 60歳以上の親・祖父母から18歳以上の子・孫へ | 相続時に贈与分を合算して課税される。年110万円の基礎控除あり |
| 教育資金の一括贈与 | 1,500万円まで | 30歳未満の子・孫へ、教育目的 | 金融機関を通じた手続きが必要である。期限付きの特例措置 |
| 結婚・子育て資金 | 1,000万円まで | 18歳以上50歳未満の子・孫へ | 金融機関を通じた手続きが必要である。期限付きの特例措置 |
| 住宅取得等資金 | 省エネ等住宅：1,000万円、一般住宅：500万円 | 18歳以上の子・孫へ、住宅取得目的 | 住宅の省エネ性能等により非課税枠が異なる。適用期限の確認が必要である |

※ 金額や条件は税制改正により変更されることがある。最新の情報は税理士または国税庁のウェブサイトで確認を促す。

> **出典**: 国税庁「No.4508 直系尊属から住宅取得等資金の贈与を受けた場合の非課税」 https://www.nta.go.jp/taxes/shiraberu/taxanswer/sozoku/4508.htm

### 3-6. 成年後見制度

▶ 提示条件：相談者が「認知症の親の財産管理が不安」「後見制度について知りたい」と述べた場合に提示する。

親の判断能力が低下した場合に、財産管理や契約行為を支援する制度である。大きく2つの種類がある。

| 項目 | 任意後見 | 法定後見 |
|---|---|---|
| 開始のタイミング | 判断能力があるうちに契約 | 判断能力が低下した後に申し立て |
| 後見人の選び方 | 本人が信頼できる人を選ぶ | 家庭裁判所が選任する |
| 手続き | 公証役場で任意後見契約を結ぶ | 家庭裁判所に申し立てる |
| 費用の目安 | 契約時: 数万円程度、報酬: 取り決めによる | 申し立て費用: 1〜2万円程度、専門家が後見人の場合: 月2〜6万円程度 |
| 柔軟性 | 本人の意向を反映しやすい | 裁判所の監督のもとで行われる |

**相談者への伝え方のポイント**

- 「親御さんの意思を尊重するためにも、判断能力がしっかりしているうちに任意後見を検討されるのは良い選択肢です」
- 「法定後見は必要に迫られてからの手続きになるため、できれば早めの備えをおすすめします」
- 手続きの詳細は司法書士や弁護士への相談を促す

### 3-7. 遺産分割協議の進め方

▶ 提示条件：相談者が「遺産の分け方で困っている」「遺産分割協議の流れを知りたい」と述べた場合に提示する。

遺言書がない場合、法定相続人全員で話し合って分け方を決める必要がある。

**基本的な流れ**

1. 法定相続人を確定する（戸籍謄本の収集）
2. 相続財産の全容を把握する（不動産、預貯金、有価証券、負債など）
3. 全員で話し合い、分け方を決める
4. 遺産分割協議書を作成する（全員の署名・実印が必要）
5. 協議書に基づいて名義変更等の手続きを行う

**注意点**

- 法定相続人が一人でも欠けた協議は無効になる
- 話し合いがまとまらない場合は、家庭裁判所の調停を利用できる
- 不動産の分割は「現物分割」「代償分割」「換価分割」「共有」の方法がある

### 3-8. 相続放棄

▶ 提示条件：相談者が「借金があるかもしれない」「相続放棄について知りたい」と述べた場合に提示する。

相続が発生した際、借金などの負債が多い場合は相続放棄を選択できる。

- **期限**: 相続の開始があったことを知った日から3ヶ月以内
- **手続き先**: 家庭裁判所
- **効果**: 初めから相続人ではなかったものとみなされる（プラスの財産も受け取れない）
- **注意点**: 一度放棄すると原則として撤回できない。期限内に判断が難しい場合は、家庭裁判所に期間の伸長を申し立てることができる
