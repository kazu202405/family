# 介護のはじめかた -- AIカウンセラー用ナレッジベース

> **対象ユーザー**: 高齢の親を心配する40〜60代の家族
> **スキル領域**: 介護保険制度、要介護認定、ケアマネジャー、在宅介護サービス
> **最終更新日**: 2026-03-23
> **出典**: 厚生労働省「介護保険制度の概要」、各自治体の介護保険ガイドライン

---

## 1. このスキルの対象となる相談例

以下のような言葉が含まれる相談に対して、このナレッジを参照すること。

1. 「親が最近よく転ぶようになって、このまま一人暮らしで大丈夫かなと思って...」
2. 「介護保険って聞いたことはあるけど、何から始めればいいのかまったくわからない」
3. 「母が入院して、来週退院と言われたけど、自宅に帰れる状態じゃない気がする」
4. 「ケアマネさんって誰に頼めばいいんですか？どうやって探すんですか？」
5. 「仕事を辞めないと親の面倒は見られないのかなと思うと、正直つらい」
6. 「デイサービスを使いたいけど、親が"まだそんなの必要ない"って怒る」
7. 「兄弟がいるのに介護は全部私に押し付けられている気がする」
8. 「施設に入れるのは親に悪いと思ってしまう...でも限界で...」

## 2. カウンセリングの心得

### 2-1. この分野特有の心理的配慮

介護の相談者は、以下のような感情を同時に抱えていることが多い。一つの感情だけを取り上げず、複数の気持ちが混在していることを前提に対応する。

| 心理状態 | 背景 | 対応の方向性 |
|---|---|---|
| 初めての介護への恐怖 | 何をすればいいかわからない、終わりが見えない | 「最初の一歩」だけに焦点を絞り、全体像は段階的に伝える |
| 自分の生活との両立不安 | 仕事・子育て・自分の健康との板挟み | 「全部を一人で抱えなくていい」と伝え、使える制度を具体的に示す |
| 「まだ早い」という否認 | 親の老いを認めたくない気持ち | 否認を否定せず、「今のうちにできる準備」として情報を提供する |
| 親への申し訳なさ | 施設利用やサービス導入への罪悪感 | プロの力を借りることは「見捨てる」こととは違うと伝える |
| きょうだい間の不公平感 | 介護負担の偏り、連絡しても動かない家族 | 感情を受け止めたうえで、分担を「交渉」として整理する |

### 2-2. 使ってはいけない言葉・表現

以下の表現は相談者を追い詰める可能性があるため、絶対に使用しない。

| 使ってはいけない表現 | なぜ問題か | 言い換え例 |
|---|---|---|
| 「介護は大変ですよ」 | 不安をあおり、相談者を萎縮させる | 「わからないことが多いと不安になりますよね」 |
| 「もっと早く相談すべきでした」 | 責められていると感じさせる | 「今気づいて動き出せたこと自体が大事な一歩です」 |
| 「お子さんの義務ですから」 | 義務感で追い詰める | 「できる範囲で関わり方を一緒に考えましょう」 |
| 「お母さん/お父さんがかわいそう」 | 罪悪感を強める | 「ご本人にとっても安心できる方法を探しましょう」 |
| 「施設に入れたほうがいいですよ」 | 決断を急がせる、罪悪感を生む | 「在宅と施設、それぞれの良さを一緒に整理してみましょうか」 |
| 「みんなやっていますよ」 | 個別の苦しさを軽視する | 「それぞれの家庭に事情がありますから、あなたに合う方法を考えましょう」 |

### 2-3. 共感のテンプレート

状況に応じて以下の共感表現を活用する。機械的にならないよう、相談者の具体的な言葉を引用しながら使うこと。

- **不安への共感**: 「初めてのことで不安を感じるのは自然なことです。わからないことを一つずつ整理していきましょう。」
- **疲弊への共感**: 「ここまでお一人で頑張ってこられたんですね。プロの力を借りることは、決して手抜きではありません。」
- **罪悪感への共感**: 「親御さんのことを大切に思っているからこそ悩まれるんですね。"完璧な介護"を目指さなくても大丈夫です。」
- **怒りへの共感**: 「ご家族の協力が得られないと、孤独に感じますよね。その気持ちはとても自然なことです。」
- **迷いへの共感**: 「どちらが正解かすぐには決められなくて当然です。判断に必要な情報を一緒に集めていきましょう。」

## 3. 基礎知識

### 3-1. 介護保険制度の全体像

介護保険は、介護が必要になった方を社会全体で支える公的な保険制度。

- **保険料を払う人（被保険者）**:
  - **第1号被保険者**: 65歳以上のすべての方。原因を問わず、介護が必要と認定されればサービスを利用できる
  - **第2号被保険者**: 40〜64歳の医療保険加入者。特定疾病（末期がん、関節リウマチ、脳血管疾患など16疾病）が原因の場合に利用できる
- **サービスの利用**: 要介護（要支援）認定を受けることで、所得に応じた自己負担（原則1〜3割）で各種サービスを利用可能
- **運営主体**: 市区町村が保険者として運営
- **財源**: 保険料（50%）と公費・税金（50%）で構成

> **相談者への伝え方のコツ**: 「介護保険は40歳から保険料を納めている制度なので、"お世話になる"のではなく、"自分が積み立ててきた制度を使う"という感覚で大丈夫です。」

### 3-2. 要介護認定の申請フロー

介護サービスを利用するためには、まず「要介護認定」を受ける必要がある。

```
申請 → 訪問調査 → 主治医意見書 → 審査判定 → 結果通知
```

| ステップ | 内容 | 所要期間の目安 |
|---|---|---|
| 1. 申請 | 市区町村の介護保険課または地域包括支援センターの窓口で申請書を提出。本人・家族・ケアマネジャーなどが代理申請も可能 | 当日 |
| 2. 訪問調査 | 市区町村の認定調査員が自宅や病院を訪問し、心身の状態を74項目で調査する。家族の立ち会いが望ましい | 申請から1〜2週間後 |
| 3. 主治医意見書 | 市区町村から主治医へ依頼。かかりつけ医がいない場合は市区町村が指定する医師の診察を受ける | 訪問調査と並行 |
| 4. 一次判定 | 訪問調査の結果をコンピュータで分析し、介護に必要な時間（要介護認定等基準時間）を算出 | -- |
| 5. 二次判定（介護認定審査会） | 一次判定結果・主治医意見書・特記事項をもとに、保健・医療・福祉の専門家が合議で最終判定 | -- |
| 6. 結果通知 | 認定結果が郵送で届く。「非該当」「要支援1〜2」「要介護1〜5」のいずれかに区分される | 申請から原則30日以内 |

> **補足**: 認定結果に不服がある場合は、60日以内に都道府県の介護保険審査会に審査請求できる。また、状態が変わった場合は「区分変更申請」をいつでも出せる。

> **相談者への伝え方のコツ**: 「申請してから結果が届くまでおおむね1か月かかりますが、急を要する場合は申請日からサービスを暫定的に使い始められる場合もあります。まずは申請を出すことが大事です。」

### 3-3. 要介護度の目安

認定された区分によって、1か月に利用できるサービスの上限額（区分支給限度基準額）が異なる。以下は目安であり、個人の状態により実際の認定は異なる場合がある。

| 区分 | 状態の目安 | 区分支給限度基準額（月額） |
|---|---|---|
| 要支援1 | 日常生活はほぼ自立しているが、一部の動作に見守りや手助けが必要。介護予防の取り組みで改善が見込まれる | 約50,320円 |
| 要支援2 | 要支援1より日常生活動作の能力がやや低下。立ち上がりや歩行に不安定さがある | 約105,310円 |
| 要介護1 | 立ち上がり・歩行が不安定。排泄・入浴に一部介助が必要。認知機能の低下が見られることもある | 約167,650円 |
| 要介護2 | 立ち上がり・歩行が自力で困難。排泄・入浴に介助が必要。日常生活全般に見守りや手助けが必要 | 約197,050円 |
| 要介護3 | 立ち上がり・歩行が自力ではできない。排泄・入浴・着替えに全面的な介助が必要。認知症の症状が見られることもある | 約270,480円 |
| 要介護4 | 日常生活全般で全面的な介助が必要。意思の伝達が困難になることもある。介護なしでは生活が困難 | 約309,380円 |
| 要介護5 | 寝たきりに近い状態。生活全般にわたって全面的な介助が必要。意思の伝達がほとんどできない | 約362,170円 |

> **注意**: 上記の金額は国が定める基準額の目安。自己負担は所得に応じて1〜3割。限度額を超えた分は全額自己負担になる。金額は改定されることがあるため、最新の情報は市区町村の窓口で確認すること。

### 3-4. ケアマネジャー（介護支援専門員）の役割・選び方・変え方

#### 役割

ケアマネジャーは、介護サービスを利用するための「相談窓口」であり「調整役」。

- 相談者の状況を聞き取り、ケアプラン（介護サービス計画）を作成する
- サービス事業者との連絡調整を行う
- 定期的に自宅を訪問し、サービスが適切かどうかモニタリングする
- 要介護認定の更新手続きを代行してくれることもある
- ケアマネジャーへの報酬は全額介護保険から支払われるため、利用者の自己負担はない

#### 選び方のポイント

- **地域包括支援センターに相談する**: 自宅近くの居宅介護支援事業所（ケアマネが所属する事業所）のリストをもらえる
- **複数の事業所に問い合わせてみる**: 対応の丁寧さや相性を比較する
- **確認したいこと**:
  - 担当件数が多すぎないか（目安: 1人あたり35件以下が望ましい）
  - 医療系の知識があるか（看護師や理学療法士の資格を持つケアマネもいる）
  - 連絡のつきやすさ（緊急時の対応体制）
  - 自宅からの距離（訪問のしやすさ）

#### 変更の手順

ケアマネジャーは理由を問わずいつでも変更できる。

1. 新しいケアマネジャー（居宅介護支援事業所）を先に見つける
2. 新しいケアマネに「変更したい」と伝えれば、引き継ぎの段取りを進めてくれる
3. 現在のケアマネに直接伝えにくい場合は、地域包括支援センターに仲介を依頼できる
4. 変更に伴う費用負担はない

> **相談者への伝え方のコツ**: 「ケアマネさんとの相性は大事です。"合わないな"と感じたら変更して大丈夫です。変えたからといって不利になることはありません。」

### 3-5. 在宅介護サービス一覧

要介護認定を受けると、以下のような在宅サービスを利用できる。費用は介護保険適用で原則1〜3割の自己負担。

| サービス名 | 内容 | 費用の目安（1割負担の場合） |
|---|---|---|
| **訪問介護（ホームヘルプ）** | ヘルパーが自宅を訪問し、身体介護（入浴・排泄・食事の介助）や生活援助（掃除・洗濯・買い物）を行う | 身体介護30分〜1時間: 約400円、生活援助45分以上: 約230円 |
| **訪問看護** | 看護師が自宅を訪問し、医療的ケア（バイタルチェック、服薬管理、医療処置、ターミナルケア）を行う | 30分〜1時間: 約820円 |
| **訪問リハビリテーション** | 理学療法士・作業療法士が自宅を訪問し、機能訓練を行う | 1回（20分）: 約310円 |
| **通所介護（デイサービス）** | 日中に施設に通い、食事・入浴・機能訓練・レクリエーションを受ける。送迎付きが一般的 | 7〜8時間/日: 約650〜900円（規模・介護度による）+食費（自費: 約500〜800円） |
| **通所リハビリテーション（デイケア）** | 医療機関や介護老人保健施設に通い、医師の指示のもとリハビリを受ける | 6〜8時間/日: 約700〜1,100円（介護度による） |
| **短期入所（ショートステイ）** | 施設に短期間宿泊し、介護やリハビリを受ける。家族の休息（レスパイト）目的でも利用可能 | 1泊: 約600〜1,000円（介護度による）+食費・滞在費（自費: 約1,500〜2,500円/日） |
| **福祉用具貸与（レンタル）** | 車いす、介護ベッド、歩行器、手すりなどを月額レンタルできる | 月額の1〜3割（例: 車いす月額約300〜500円、介護ベッド月額約700〜1,300円） |
| **福祉用具購入** | 入浴補助用具、ポータブルトイレ、簡易浴槽など衛生面から貸与になじまない用具を購入できる | 年間10万円まで保険適用（自己負担1〜3割） |
| **住宅改修** | 手すりの取付け、段差の解消、引き戸への変更、和式から洋式トイレへの変更などの改修費用に保険適用 | 上限20万円まで保険適用（自己負担1〜3割）。原則1回限り |

> **注意**: 上記の費用はあくまで概算の目安。地域やサービス提供事業者、加算の有無によって異なる。正確な金額はケアマネジャーに確認すること。

### 3-6. 地域密着型サービス

住み慣れた地域で暮らし続けるために、市区町村が指定・監督するサービス。原則として、その市区町村に住民票がある方が利用対象。

| サービス名 | 内容 |
|---|---|
| **小規模多機能型居宅介護** | 「通い」「訪問」「泊まり」を1つの事業所で柔軟に組み合わせて利用できる。顔なじみのスタッフが対応する安心感がある |
| **定期巡回・随時対応型訪問介護看護** | 24時間対応で、日中・夜間を通じて定期的な訪問と緊急時の随時対応を行う。一人暮らしの方に向いている |
| **認知症対応型通所介護（認知症デイ）** | 認知症の方を対象に少人数（定員12名以下）できめ細かいケアを行うデイサービス |
| **地域密着型介護老人福祉施設（小規模特養）** | 定員29名以下の小規模な特別養護老人ホーム。地域の中で家庭的な雰囲気で暮らせる |
| **看護小規模多機能型居宅介護（かんたき）** | 小規模多機能型に訪問看護を加えたサービス。医療ニーズの高い方にも対応できる |

## 4. よくある状況と対応パターン

### 4-1. 「親が急に入院して退院が迫っている」

**相談者の心理**: パニックに近い状態。何をすればいいかわからない。時間がない焦り。

**対応の流れ**:

1. **まず落ち着いてもらう**: 「退院までに全部決めなくても大丈夫です。まず今日できることを整理しましょう。」
2. **病院の医療ソーシャルワーカー（MSW）に相談する**: 退院後の生活について相談できる専門職が病院にいることを伝える。MSWがいない場合は看護師長や退院支援看護師に声をかける。
3. **要介護認定を申請する**: 入院中でも申請できる。病院で訪問調査を受けることも可能。申請は家族が代理で行える。
4. **暫定ケアプランの活用**: 認定結果が出る前でも、暫定のケアプランを立ててサービスを利用開始できる場合がある。ケアマネジャーに相談すること。
5. **退院前カンファレンスへの参加**: 病院スタッフ・ケアマネ・サービス事業者が集まり、退院後の生活を計画する場。家族も参加できるので、不安な点を伝える。

### 4-2. 「まだ元気だけど足腰が弱ってきた」

**相談者の心理**: 「まだ介護は早い」と思いつつ、漠然とした不安がある。

**対応の流れ**:

1. **不安を受け止める**: 「"今のうちに何かできることはないか"と考えること自体がとても大切です。」
2. **地域包括支援センターへの相談を提案する**: 要介護認定を受けていなくても相談できる。介護予防の教室や見守りサービスなどを紹介してもらえる。
3. **介護予防・日常生活支援総合事業（総合事業）の案内**: 要支援や「基本チェックリスト」に該当する方が利用できるサービス。体操教室、配食サービス、通いの場など。
4. **住環境の見直し**: 転倒予防のための手すり設置や段差解消は、要介護認定を受ければ介護保険の住宅改修費が使える。認定前でも自治体独自の助成がある場合も。
5. **かかりつけ医の確認**: いざというとき主治医意見書が必要になるため、定期的に受診しているかを確認しておく。

### 4-3. 「仕事と介護の両立ができない」

**相談者の心理**: 仕事を辞めなければならないかもしれないという恐怖。経済的不安。孤立感。

**対応の流れ**:

1. **「仕事を辞めないで済む方法を一緒に考えましょう」と伝える**: 介護離職は経済的・精神的なリスクが大きい。辞める前に使える制度を確認することが大切。
2. **介護休業制度の案内**:
   - 対象家族1人につき通算93日まで、3回を上限に分割取得可能
   - 雇用保険から「介護休業給付金」（休業前賃金の67%）が支給される
   - 申請先は勤務先の人事・総務部門
3. **介護休暇の案内**:
   - 対象家族1人につき年5日（2人以上なら年10日）取得可能
   - 時間単位での取得も可能
   - 通院の付き添い、ケアマネとの面談などに使える
4. **勤務時間の柔軟化**: 短時間勤務制度、フレックスタイム、時差出勤、テレワークなど、事業主に措置義務・努力義務がある制度を案内する
5. **介護サービスの最大活用**: デイサービス、訪問介護、ショートステイを組み合わせることで、日中は仕事に集中できる体制を作れる場合がある

> **相談者への伝え方のコツ**: 「介護のために仕事を辞めると、経済的にも精神的にもご自身が追い詰められやすくなります。"辞めずに続ける方法"を先に探してみましょう。」

### 4-4. 「在宅と施設どちらがいいか迷っている」

**相談者の心理**: 施設に入れることへの罪悪感と、在宅介護の限界への不安。

**判断のフレームワーク（以下の観点を一緒に整理する）**:

| 観点 | 在宅が向いている場合 | 施設が向いている場合 |
|---|---|---|
| 本人の意向 | 住み慣れた家にいたい | 一人の時間が不安、人と交流したい |
| 介護度 | 要介護1〜2程度で在宅サービスで対応できる | 要介護3以上で常時介護が必要 |
| 家族の介護力 | 同居または近居の家族がサポートできる | 遠距離、仕事との両立が困難 |
| 医療ニーズ | 医療的ケアが少ない | 夜間の医療対応が必要 |
| 経済面 | 在宅サービスの範囲で収まる | 特養（原則要介護3以上）なら月額5〜15万円程度で利用可 |
| 認知症の程度 | 軽度で見守りで対応できる | 徘徊や火の不始末があり安全確保が難しい |

> **相談者への伝え方のコツ**: 「どちらが"正解"ということはありません。ご本人の気持ちとご家族の状況の両方を大切にして、"今の時点でのベスト"を一緒に考えましょう。状況が変われば、選択を変えることもできます。」

### 4-5. 「ケアマネさんと合わない」

**相談者の心理**: 不満はあるが言い出しにくい。変えたら不利になるのではという不安。

**対応の流れ**:

1. **気持ちを受け止める**: 「合わないと感じることは珍しくありません。我慢し続ける必要はないですよ。」
2. **具体的な不満を整理する**: 連絡が遅い、希望を聞いてくれない、説明が不十分、提案が少ないなど
3. **まずは本人に伝えてみることを提案**: 改善の余地がある場合もある
4. **変更の手順を説明する**:
   - 新しい居宅介護支援事業所を探す（地域包括支援センターに相談可）
   - 新しいケアマネに「変更希望」と伝えれば引き継ぎを調整してくれる
   - 現ケアマネに直接言いにくい場合は、地域包括支援センターに仲介を頼める
   - サービスの利用が途切れることはない
   - 費用はかからない

## 5. 相談先ガイド

### 5-1. 地域包括支援センター

**どんなとき**: 介護が必要かどうかわからない段階から相談できる。要介護認定前の方の「最初の相談窓口」。

- 保健師・社会福祉士・主任ケアマネジャーが在籍
- 介護予防ケアプランの作成（要支援の方）
- 高齢者の権利擁護（虐待防止、成年後見制度の相談）
- 地域の介護サービスや医療機関の情報提供

**持っていくと話が早いもの**:
- 介護保険被保険者証（65歳以上の方に交付済み）
- 気になる症状や日常生活の困りごとのメモ
- かかりつけ医の情報（病院名・医師名）
- 服用中の薬の情報（お薬手帳）

**聞かれること**:
- ご本人の年齢、住所、家族構成
- 日常生活で困っていること（具体的な場面）
- 現在の病気や通院状況
- 家族の介護への関わり方

### 5-2. 居宅介護支援事業所

**どんなとき**: 要介護1〜5の認定を受けた方がケアマネジャーを探すとき。

- ケアマネジャーが所属する事業所
- ケアプランの作成と管理
- サービス事業者との連絡調整

**探し方**:
- 地域包括支援センターから紹介してもらう
- 市区町村のホームページや広報に掲載されている事業所一覧を確認
- 「介護サービス情報公表システム」（厚生労働省）で検索

### 5-3. 医療ソーシャルワーカー（MSW）

**どんなとき**: 入院中の親の退院後の生活について相談したいとき。

- 病院に配置されている生活相談の専門職
- 退院後の介護サービスの手配や施設探しを支援
- 医療費や生活費の相談にも対応

**聞かれること**:
- 退院後の生活環境（自宅の構造、同居家族の有無）
- ご本人・家族の希望（在宅か施設か）
- 経済的な状況

**声のかけ方**: 「退院後の生活のことで相談したいのですが、ソーシャルワーカーさんとお話しできますか？」とナースステーションに伝える。

### 5-4. 市区町村の介護保険課

**どんなとき**: 要介護認定の申請、保険料の相談、制度全般の問い合わせ。

- 要介護認定の申請受付
- 介護保険料の納付・減免の相談
- 介護保険証の再発行

**持っていくと話が早いもの**:
- 介護保険被保険者証
- 本人確認書類（マイナンバーカード、運転免許証など）
- 主治医の情報（要介護認定申請の場合）

## 6. 家族への伝え方

### 6-1. 親に「介護サービスを使おう」と伝えるコツ

高齢の親は「まだ自分でできる」「人の世話にはなりたくない」と感じていることが多い。以下のアプローチを提案する。

- **「介護」という言葉を使わない**: 「健康づくりの教室があるんだって」「お風呂が安全に入れるようになるサービスがあるよ」など、具体的なメリットを伝える
- **主治医やかかりつけ医から勧めてもらう**: 家族の言葉より医師の言葉のほうが受け入れやすい場合がある
- **体験利用を提案する**: 「一度だけ試してみない？合わなければやめればいいから」
- **本人の「困りごと」に焦点を当てる**: 「お風呂で滑りそうで怖いって言ってたよね？それを解決できるかもしれないよ」
- **一度に全部を変えようとしない**: 小さなサービスから始めて、徐々に慣れてもらう

### 6-2. きょうだい間での介護分担の切り出し方

- **事実ベースで話す**: 「今こういう状況で、こういう支援が必要になっている」と客観的な情報を共有する
- **「責める」のではなく「相談する」姿勢で**: 「どうやって分担できるか一緒に考えたい」と伝える
- **役割を具体的に提案する**: 「月に1回の通院付き添いをお願いできないか」「費用の一部を負担してもらえないか」など
- **遠方のきょうだいにもできることを提示する**: 費用の分担、電話での定期連絡、帰省時のレスパイト（介護者の休息）、手続きの代行など
- **家族会議の場を設ける**: 対面が難しければビデオ通話でも可。議事録を残すと後々のトラブルを防げる

### 6-3. 「自分だけが背負っている」と感じるときの対処

- **その感情は正当であることを伝える**: 「不公平だと感じるのは当然です。」
- **介護者自身のケアを最優先に**: 介護者が倒れたら介護は成り立たない
- **使える外部サービスを最大限活用する**: ショートステイで定期的に休む、訪問介護で日常の負担を減らす
- **介護者の会への参加を提案する**: 同じ立場の人と話すことで気持ちが楽になることがある。地域包括支援センターで情報を得られる
- **ケアマネジャーに「自分がつらい」と伝えてよいことを助言する**: ケアマネは介護者の状態も考慮してプランを調整できる

## 7. 制度・法的情報

### 7-1. 介護保険の自己負担割合

介護サービスを利用したときの自己負担割合は、本人の所得に応じて決まる。

| 自己負担割合 | 対象となる方 |
|---|---|
| **1割** | 本人の合計所得金額が160万円未満の方（大多数の方が該当） |
| **2割** | 本人の合計所得金額が160万円以上220万円未満の方（ただし、年金収入+その他所得が単身で280万円未満、夫婦で346万円未満の場合は1割） |
| **3割** | 本人の合計所得金額が220万円以上の方（ただし、年金収入+その他所得が単身で340万円未満、夫婦で463万円未満の場合は2割） |

> **出典**: 厚生労働省「介護保険の利用者負担割合について」

### 7-2. 高額介護サービス費

1か月の自己負担額が上限を超えた場合、超えた分が後から払い戻される制度。

| 所得区分 | 月額上限（世帯合算） |
|---|---|
| 生活保護受給者 | 15,000円（個人） |
| 住民税非課税世帯（年金収入80万円以下等） | 15,000円（個人）/ 24,600円（世帯） |
| 住民税非課税世帯 | 24,600円 |
| 住民税課税〜課税所得380万円未満 | 44,400円 |
| 課税所得380万円〜690万円未満 | 93,000円 |
| 課税所得690万円以上 | 140,100円 |

申請は市区町村の介護保険課で行う。一度申請すれば、以降は自動的に払い戻される自治体が多い。

> **出典**: 厚生労働省「高額介護サービス費の負担限度額」

### 7-3. 介護休業法の概要（育児・介護休業法）

| 項目 | 内容 |
|---|---|
| **介護休業** | 対象家族1人につき通算93日まで、3回を上限に分割取得可能 |
| **介護休暇** | 対象家族1人につき年5日（2人以上なら年10日）。時間単位取得可 |
| **所定外労働の制限** | 事業主に請求すれば残業を免除してもらえる |
| **時間外労働の制限** | 月24時間、年150時間を超える時間外労働の免除を請求可能 |
| **深夜業の制限** | 午後10時〜午前5時の勤務免除を請求可能 |
| **短時間勤務等の措置** | 事業主は短時間勤務・フレックス・時差出勤・費用助成のいずれかの措置を講じる義務がある |
| **介護休業給付金** | 雇用保険から休業前賃金の67%が支給される。申請先はハローワーク |
| **不利益取扱いの禁止** | 介護休業等を理由とする解雇・降格・減給等は法律で禁止されている |

> **出典**: 厚生労働省「育児・介護休業法のあらまし」

### 7-4. その他の経済的支援制度

- **特定入所者介護サービス費（補足給付）**: 住民税非課税世帯の方が施設入所・ショートステイを利用する際、食費・居住費の負担が軽減される制度
- **医療費控除**: 一定の介護サービスの自己負担分は確定申告で医療費控除の対象になる
- **障害者控除対象者認定**: 要介護認定を受けた方が一定の条件を満たすと、所得税・住民税の障害者控除を受けられる場合がある。市区町村に申請が必要

## 補足: このナレッジを使うときの注意

- 制度の金額や基準は法改正により変わることがある。具体的な金額を回答するときは「目安」であることを必ず伝え、「最新の情報は市区町村の窓口やケアマネジャーに確認してください」と添えること
- 個別の医療判断（「施設に入るべきか」「このサービスが合っているか」など）は行わない。専門職への相談を促すこと
- 相談者が精神的に追い詰められている兆候がある場合は、介護者向けの相談窓口（地域包括支援センター、よりそいホットライン 0120-279-338）への連絡を案内すること
- **基本姿勢**: 「完璧にやらなくていい」「プロに頼っていい」「一人で抱え込まなくていい」をすべての回答の根底に置くこと
