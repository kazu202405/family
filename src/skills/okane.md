# お金と制度に関する専門知識

> AIカウンセラー向けナレッジベース
> 対象ユーザー: 高齢の親を心配する40〜60代の家族
> 最終更新日: 2026年3月23日
> 主な出典: 厚生労働省、国税庁、日本年金機構の公開資料に基づく

---

## 1. このスキルの対象となる相談例

以下のような相談が寄せられた場合、このスキルを参照して回答する。

1. 「介護にいくらかかるの？ 全体の費用が見えなくて不安です」
2. 「親の年金だけで施設に入れますか？」
3. 「介護費用を兄弟で分担したいけど、どう切り出せばいいですか？」
4. 「高額介護サービス費っていう制度があると聞いたけど、うちは使えますか？」
5. 「世帯分離すると介護費用が安くなるって本当ですか？」
6. 「親の貯蓄が減ってきていて、このままだと足りなくなりそうです」
7. 「介護の費用って確定申告で戻ってくるんですか？」
8. 「自分の老後資金も心配だけど、親の介護費用も出さないといけない」

## 2. カウンセリングの心得

### お金の不安の本質を理解する

- お金の不安は「生活が成り立たなくなるかもしれない」という根源的な恐怖に直結する。軽く扱わず、深刻に受け止めること
- 相談者は「お金の話は恥ずかしい」「家庭の経済状況を他人に話したくない」と感じていることが多い。打ち明けてくれたこと自体に敬意を払う
- 介護費用の分野は「知らないと損する制度」が非常に多い。正確な情報提供そのものが大きな支援になる

### 絶対に使ってはいけない言葉

| 使ってはいけない表現 | 理由 |
|---|---|
| 「お金がないと厳しいですね」 | 追い詰める表現であり、相談者の無力感を強める |
| 「貯蓄が足りないですね」 | 自己責任を暗に問う表現になる |
| 「子どもが負担すべきです」 | 家族の義務を押しつける表現であり、相談者を追い込む |
| 「なぜもっと早く相談しなかったのですか」 | 過去を責めても解決にはつながらない |
| 「普通はこのくらい貯めています」 | 比較は不安を増幅させるだけ |

### 使うべき言い換え

| 場面 | 推奨する表現 |
|---|---|
| 費用面の不安を受け止めるとき | 「不安になるのは当然のことです。使える制度を一緒に整理していきましょう」 |
| 負担軽減の提案をするとき | 「負担を減らす方法がいくつかありますので、順番に確認していきましょう」 |
| 制度を知らなかったと言われたとき | 「今知ることができてよかったです。これから活用していきましょう」 |
| 兄弟間の費用分担の悩みに対して | 「費用の分担は多くのご家族が悩まれるところです。いくつかの考え方をご紹介しますね」 |

### 回答時の基本姿勢

- 金額は必ず「目安」「一般的には」を付けて伝える。個別の状況で大きく変わるため断定しない
- 制度は年度によって変更される可能性があることを必ず伝える
- 具体的な判断が必要な場面では「まずは窓口で確認されることをおすすめします」と添える
- 「知らなかった」ことを責めない。「今知れてよかったですね」と前向きに

## 3. 基礎知識

### 3-1. 介護にかかる費用の全体像

介護費用は「在宅介護」と「施設介護」で大きく異なる。相談者が漠然と不安を抱えている場合は、まずこの全体像を示すことで見通しを持ってもらう。

#### 在宅介護の月額費用の目安

| 項目 | 月額の目安 |
|---|---|
| 介護保険サービス自己負担分 | 1万〜3万円程度（1割負担の場合） |
| 介護保険外の費用（おむつ代、日用品など） | 1万〜3万円程度 |
| 住宅改修・福祉用具（初期費用を月割り） | 数千円程度 |
| 合計 | 月3万〜8万円程度 |

※要介護度、利用するサービスの種類・頻度によって大きく変動する。

#### 施設介護の月額費用の目安

| 施設の種類 | 月額の目安 |
|---|---|
| 特別養護老人ホーム（特養） | 5万〜15万円程度 |
| 介護老人保健施設（老健） | 8万〜15万円程度 |
| グループホーム | 12万〜18万円程度 |
| 介護付き有料老人ホーム | 15万〜35万円程度 |
| サービス付き高齢者向け住宅 | 10万〜25万円程度 |

※上記は食費・居住費を含む目安。入居一時金が必要な施設もある。特養は所得に応じた減額制度（特定入所者介護サービス費）がある。

#### 介護の初期費用の目安

- 住宅改修（手すり設置、段差解消など）: 介護保険で上限20万円まで9割給付
- 福祉用具購入: 介護保険で年間上限10万円まで9割給付
- その他（介護ベッド、車いすなど）: レンタル制度あり（月数百円〜数千円の自己負担）

### 3-2. 介護保険の自己負担割合

介護保険サービスの自己負担割合は、本人の所得によって1割・2割・3割のいずれかに決まる。

| 自己負担割合 | 本人の合計所得金額の目安 |
|---|---|
| 1割 | 160万円未満 |
| 2割 | 160万円以上220万円未満（単身で年金収入のみの場合、年収280万円以上が目安） |
| 3割 | 220万円以上（単身で年金収入のみの場合、年収340万円以上が目安） |

※65歳以上の第1号被保険者が対象。合計所得金額には年金以外の所得も含む。同じ世帯の65歳以上の方の年金収入やその他の所得も判定に影響する場合がある。

### 3-3. 高額介護サービス費

月々の介護サービスの自己負担が一定額を超えた場合、超過分が払い戻される制度。

| 区分 | 月額上限額 |
|---|---|
| 生活保護受給者 | 15,000円（個人） |
| 世帯全員が住民税非課税で、年金収入80万円以下等 | 15,000円（個人）/ 24,600円（世帯） |
| 世帯全員が住民税非課税 | 24,600円（世帯） |
| 住民税課税〜課税所得380万円未満 | 44,400円（世帯） |
| 課税所得380万円以上690万円未満 | 93,000円（世帯） |
| 課税所得690万円以上 | 140,100円（世帯） |

**申請方法**: 市区町村の介護保険課に申請。初回申請後は自動的に払い戻しが続く自治体が多い。該当する場合は市区町村から通知が届くことが多いが、届かない場合は自分から問い合わせること。

### 3-4. 高額医療・高額介護合算療養費制度

同じ世帯で、1年間（毎年8月〜翌年7月）の医療保険と介護保険の自己負担を合算して、上限額を超えた分が払い戻される制度。

| 区分（70歳以上の場合） | 年間上限額 |
|---|---|
| 住民税非課税（低所得I） | 19万円 |
| 住民税非課税（低所得II） | 31万円 |
| 一般（課税所得145万円未満） | 56万円 |
| 現役並みI（課税所得145万円以上） | 67万円 |
| 現役並みII（課税所得380万円以上） | 141万円 |
| 現役並みIII（課税所得690万円以上） | 212万円 |

**申請方法**: 加入している医療保険と市区町村の介護保険課の両方に申請が必要。基準日（毎年7月31日）に加入していた医療保険に申請する。

### 3-5. 医療費控除（介護サービスの対象範囲）

確定申告で医療費控除の対象となる介護サービスがある。

#### 対象となる介護サービス（主なもの）

- **施設サービス**: 特養の自己負担額の2分の1、老健・介護医療院の自己負担額の全額
- **居宅サービス（医療系）**: 訪問看護、訪問リハビリテーション、居宅療養管理指導、通所リハビリテーション、短期入所療養介護
- **居宅サービス（福祉系・上記医療系と併用する場合に限り対象）**: 訪問介護（生活援助中心型を除く）、通所介護、短期入所生活介護、夜間対応型訪問介護、小規模多機能型居宅介護など
- **おむつ代**: 医師が発行する「おむつ使用証明書」があれば対象。2年目以降は市区町村が発行する確認書で代替可能な場合あり

#### 控除額の計算

- 年間の医療費合計 - 保険金等で補填された額 - 10万円（または総所得金額の5%のいずれか低い方）= 医療費控除額
- 上限は200万円

### 3-6. 障害者控除対象者認定

要介護認定を受けている高齢者は、市区町村に申請することで「障害者控除対象者認定書」を取得できる場合がある。

- **対象**: 65歳以上で要介護1以上の認定を受けている方（自治体により基準が異なる）
- **控除額**: 障害者控除27万円、特別障害者控除40万円（所得税の場合）
- **効果**: 本人または扶養している家族の所得税・住民税が軽減される
- **注意**: 身体障害者手帳を持っていなくても申請できる。自治体によって認定基準が異なるため、まずは市区町村の窓口に確認すること

### 3-7. 世帯分離のメリット・デメリット・手続き

親と同居している場合、住民票上の世帯を分けることで介護費用の負担が軽減される場合がある。

#### メリット

- 高額介護サービス費の上限額が下がる可能性がある（世帯の所得で判定されるため）
- 施設入所時の食費・居住費の減額（特定入所者介護サービス費）を受けられる可能性がある
- 介護保険料が下がる場合がある

#### デメリット

- 国民健康保険に加入している場合、世帯を分けると保険料が合計で増える場合がある
- 会社の健康保険の扶養から外れる可能性がある
- 各種届出で「世帯全員の住民票」が必要な場合に手続きが煩雑になる

#### 手続き

- 市区町村の窓口で「世帯変更届」を提出する
- 届出に必要なもの: 届出人の本人確認書類、印鑑（自治体による）
- 届出理由を聞かれることがあるが、「生計を別にしているため」で問題ない

#### 判断のポイント

世帯分離が有利かどうかは個々の状況によって異なる。以下を確認したうえで判断する。

1. 現在の介護保険の自己負担段階
2. 国民健康保険への影響
3. 会社の健康保険の扶養への影響
4. 税金の扶養控除への影響

**相談者には「得になるケースも損になるケースもありますので、具体的な数字を持って市区町村の窓口やファイナンシャルプランナーに相談されることをおすすめします」と案内する。**

### 3-8. 年金の種類と受給額の目安

親の年金額は介護費用を考えるうえで重要な情報。まず大まかな仕組みを理解してもらう。

| 年金の種類 | 受給額の目安（月額） | 対象者 |
|---|---|---|
| 国民年金（老齢基礎年金） | 満額で約6.8万円 | 自営業、専業主婦（夫）など |
| 厚生年金（老齢厚生年金）+ 基礎年金 | 約14万〜16万円 | 会社員・公務員として勤務した方 |

※受給額は加入期間や報酬額によって大きく異なる。上記はあくまで一般的な目安。

#### 年金額の確認方法

- **年金振込通知書**: 毎年6月頃に届く（金額変更がない年は届かない場合あり）
- **ねんきんネット**: 日本年金機構のWebサービスで過去の記録や見込み額を確認できる
- **年金事務所**: 窓口で照会可能（委任状があれば家族でも確認できる）
- **年金額改定通知書**: 金額が変わった際に届く

### 3-9. 生活保護制度の概要（介護との関係）

年金や貯蓄だけでは介護費用を賄えない場合の最後のセーフティネット。

- **要件**: 資産の活用、能力の活用、他の制度の活用、扶養義務者の援助を行ったうえで、なお生活に困窮する場合に受給できる
- **介護扶助**: 生活保護受給者は介護保険の自己負担分が介護扶助として支給される
- **注意点**: 預貯金や不動産などの資産がある場合は原則として活用が求められる。ただし住んでいる持ち家は、一定の条件のもとで保有が認められる場合がある
- **相談先**: 市区町村の福祉事務所（生活保護の担当窓口）

**相談者に伝えるときの配慮**: 「生活保護」という言葉に抵抗感を持つ方も多い。「生活を支えるための公的な制度のひとつです」と伝え、利用すること自体は権利であることを丁寧に説明する。

### 3-10. 介護費用の確定申告での控除

介護にかかった費用の一部は、確定申告で税金の還付を受けられる場合がある。

- **医療費控除**: 3-5で記載した対象サービスの自己負担額を医療費控除として申告
- **社会保険料控除**: 親の介護保険料を子が支払った場合、子の社会保険料控除に含められる
- **扶養控除**: 親の合計所得金額が48万円以下（年金収入のみの場合158万円以下）であれば扶養控除の対象になる可能性がある
- **障害者控除**: 3-6の認定を受けていれば障害者控除を適用できる

## 4. よくある状況と対応パターン

### 4-1. 「親の年金額がわからない」

**背景**: 親がお金の話をしたがらない、離れて暮らしていて把握できていない、というケースは非常に多い。

**対応の流れ**:

1. まず、年金額がわからなくても不自然なことではないと伝えて安心してもらう
2. 確認方法を案内する
   - 親の自宅に届いている「年金振込通知書」や「年金額改定通知書」を確認
   - 通帳の振込記録から逆算する（年金は偶数月に2か月分が振り込まれる）
   - 親本人が「ねんきんネット」に登録していれば、そこで確認
   - 年金事務所に本人が行く、または委任状を持って家族が照会する
3. 年金額がわかったら、介護費用との差額を確認し、不足分をどう補うかを一緒に考える

### 4-2. 「介護費用を兄弟で分担する方法」

**背景**: 兄弟間での費用分担はトラブルになりやすく、感情的な対立に発展することも多い。

**対応の流れ**:

1. 費用分担に正解はないことを前提として共有する
2. よく使われる分担の考え方を紹介する
   - **均等割り**: 兄弟の人数で均等に分ける
   - **収入比率**: それぞれの収入に応じて割合を決める
   - **役割分担**: 近くに住む人が「時間」を、遠方の人が「お金」を出す
   - **基金方式**: 毎月一定額を共通口座に積み立て、そこから支出する
3. 話し合いの場を設けることを提案する（できれば全員が集まれるタイミングで）
4. 話し合いの前に、現在の介護費用の総額を整理しておくことを勧める
5. 合意内容は簡単でもよいので文書にしておくことを推奨する

### 4-3. 「親の貯蓄が底をつきそう」

**背景**: 切迫した不安を抱えている。冷静に使える制度を整理することが最優先。

**対応の流れ**:

1. 不安な気持ちを受け止めたうえで、使える制度を優先順位をつけて案内する
2. 制度活用の優先順位

| 優先度 | 制度 | 効果 |
|---|---|---|
| 1 | 高額介護サービス費の申請 | 月々の自己負担に上限が設けられる |
| 2 | 特定入所者介護サービス費（施設入所の場合） | 食費・居住費が大幅に減額される |
| 3 | 世帯分離の検討 | 上記の制度で有利な区分になる可能性 |
| 4 | 障害者控除対象者認定の取得 | 税負担の軽減 |
| 5 | 社会福祉協議会の生活福祉資金貸付 | 一時的な資金不足を補う低金利の貸付 |
| 6 | 生活保護の相談 | 最終的なセーフティネット |

3. 介護サービスの見直し（ケアマネジャーに相談して、過不足を調整する）も並行して提案

### 4-4. 「自分の老後資金も心配」

**背景**: 親の介護費用を負担することで、自分自身の将来に不安を感じている。

**対応の流れ**:

1. 「自分の将来も大切にしたい」という気持ちは正当であると伝える
2. 親の介護費用はまず親の年金・資産から賄うのが基本であることを確認する
3. 公的制度を最大限活用して、家族の持ち出しを最小限にする方針を整理する
4. 自分の負担が重くなりすぎないよう、限度額を決めておくことを勧める
5. 必要に応じてファイナンシャルプランナーへの相談を提案する

### 4-5. 「世帯分離した方がいいと聞いたが」

**背景**: インターネット等で情報を得て相談に来るケースが増えている。

**対応の流れ**:

1. 世帯分離のメリット・デメリットを3-7に基づいて説明する
2. 以下の判断ポイントを確認する
   - 現在の介護保険の自己負担割合と高額介護サービス費の上限額
   - 国民健康保険に加入しているか、会社の健康保険の扶養に入っているか
   - 税金上の扶養控除を利用しているか
3. 個別の試算が必要であるため、市区町村の窓口またはファイナンシャルプランナーに相談することを勧める
4. 「世帯分離は万能ではなく、状況によっては損になることもあります」と正確に伝える

## 5. 相談先ガイド

### 市区町村の介護保険課

- **相談内容**: 介護保険の手続き全般、高額介護サービス費、特定入所者介護サービス費、障害者控除対象者認定の申請
- **場所**: お住まいの市区町村の役所・役場
- **費用**: 無料
- **持ち物**: 介護保険被保険者証、本人確認書類
- **ポイント**: 制度の利用条件や申請方法について最も正確な情報が得られる窓口。まずはここに相談することを勧める

### 地域包括支援センター

- **場所**: 中学校区に1か所程度の設置。市区町村のWebサイトで管轄を確認できる
- **費用**: 無料
- **ポイント**: 介護の入口となる相談窓口。どこに相談してよいかわからない場合は、まずここに連絡する

### 年金事務所

- **相談内容**: 年金の受給額照会、受給手続き、年金記録の確認
- **場所**: 各地域に設置。日本年金機構のWebサイトで最寄りの事務所を検索できる
- **費用**: 無料
- **持ち物**: 年金手帳または基礎年金番号通知書、本人確認書類。家族が代理で行く場合は委任状が必要
- **ポイント**: 予約制の事務所が多いため、事前に電話予約することを勧める。電話での問い合わせは「ねんきんダイヤル（0570-05-1165）」も利用可能

- **費用**: 無料

### 社会福祉協議会（生活福祉資金貸付）

- **費用**: 相談は無料。貸付は低金利または無利子

## 6. 家族への伝え方

### 6-1. 親にお金の話を切り出す方法

お金の話を嫌がる親は多い。以下のアプローチを相談者に提案する。

**切り出し方の例**:

- 「自分の老後のことを考え始めたんだけど、お父さん（お母さん）はどうしてた？」と自分の話から入る
- 「介護保険の制度が変わったらしいんだけど、確認しておいた方がいいみたいだよ」と制度の話題として入る
- 「万が一のときに困らないように、年金のことだけ教えてもらっていい？」と最小限の情報から始める

**避けるべきアプローチ**:

- いきなり「貯金いくらあるの？」と聞く
- 「施設に入るお金ある？」と不安をあおる
- 兄弟全員で囲むように話す（責められていると感じさせる）

**伝え方のコツ**:

- 1対1で、落ち着いた雰囲気で話す
- 一度に全部聞こうとしない。まずは年金額だけ、など段階的に
- 「何かあったときに助けたいから」という動機を正直に伝える

### 6-2. 兄弟間での費用分担の話し合い方

**話し合いの前にやっておくこと**:

- 現在の介護費用の総額を整理する（明細があるとよい）
- 使っている公的制度と、まだ使っていない制度を把握する
- 各自の負担可能額をおおまかに考えておく

**話し合いのポイント**:

- 「お金」だけでなく「時間」「労力」も含めた総合的な分担を考える
- 近くに住む人と遠方の人で役割が異なるのは自然なこと
- 定期的（半年〜1年ごと）に見直す前提で合意する
- 合意内容は簡単なメモでよいので記録に残す

### 6-3. 「私ばかり出している」という不公平感の解消

**背景**: 介護の費用や時間の負担が一人に偏ると、強い不公平感と疲弊が生じる。

**対応の流れ**:

1. 不公平感を抱えていること自体が大きなストレスであると認める
2. お金の負担と時間の負担を見える化することを提案する
   - 月々の費用負担額をリストにする
   - 通院の付き添い、日常の見守りなどの時間負担も含める
3. 見える化したうえで、兄弟間で改めて話し合う場を設けることを勧める
4. 話し合いがまとまらない場合は、地域包括支援センターやケアマネジャーに第三者として入ってもらう方法もあることを伝える
