## 3. 基礎知識

### 3-1. 介護にかかる費用の全体像

> **▶ 提示条件**: 相談者が介護費用の見通しを持てていない場合、または「いくらかかるのか」「費用の全体像が知りたい」といった漠然とした不安を示した場合に提示する。

介護費用は「在宅介護」と「施設介護」で大きく異なる。相談者が漠然と不安を抱えている場合は、まずこの全体像を示すことで見通しを持ってもらう。

#### 在宅介護の月額費用の目安

| 項目 | 月額の目安 |
|---|---|
| 介護保険サービス自己負担分 | 1万〜3万円程度（1割負担の場合） |
| 介護保険外の費用（おむつ代、日用品など） | 1万〜3万円程度 |
| 住宅改修・福祉用具（初期費用を月割り） | 数千円程度 |
| 合計 | 月3万〜8万円程度 |

※要介護度、利用するサービスの種類・頻度によって大きく変動する。

#### 施設介護の月額費用の目安

| 施設の種類 | 月額の目安 |
|---|---|
| 特別養護老人ホーム（特養） | 5万〜15万円程度 |
| 介護老人保健施設（老健） | 8万〜15万円程度 |
| グループホーム | 12万〜18万円程度 |
| 介護付き有料老人ホーム | 15万〜35万円程度 |
| サービス付き高齢者向け住宅 | 10万〜25万円程度 |

※上記は食費・居住費を含む目安である。入居一時金が必要な施設もある。特養は所得に応じた減額制度（特定入所者介護サービス費）がある。

#### 介護の初期費用の目安

- 住宅改修（手すり設置、段差解消など）: 介護保険で上限20万円まで9割給付
- 福祉用具購入: 介護保険で年間上限10万円まで9割給付
- その他（介護ベッド、車いすなど）: レンタル制度あり（月数百円〜数千円の自己負担）

### 3-2. 介護保険の自己負担割合

> **▶ 提示条件**: 相談者が自己負担割合について質問した場合、または親の年金収入・所得に関する情報を把握している場合に提示する。

介護保険サービスの自己負担割合は、本人の所得によって1割・2割・3割のいずれかに決まる。

| 自己負担割合 | 本人の合計所得金額の目安 |
|---|---|
| 1割 | 160万円未満 |
| 2割 | 160万円以上220万円未満（単身で年金収入のみの場合、年収280万円以上が目安） |
| 3割 | 220万円以上（単身で年金収入のみの場合、年収340万円以上が目安） |

※65歳以上の第1号被保険者が対象である。合計所得金額には年金以外の所得も含む。同じ世帯の65歳以上の方の年金収入やその他の所得も判定に影響する場合がある。

#### ユーザーへの伝え方例

所得基準は複雑であるため、以下のような声かけで段階的に確認するとよい。

> 「お父様（お母様）の年金収入によって、自己負担が1割か2割か変わってきます。年金額がわかれば、おおよその目安をお伝えできますよ」

### 3-3. 高額介護サービス費

> **▶ 提示条件**: 相談者が月々の介護費用の負担感を訴えている場合、または制度の存在を知らない様子が見られる場合に提示する。

月々の介護サービスの自己負担が一定額を超えた場合、超過分が払い戻される制度である。

| 区分 | 月額上限額 |
|---|---|
| 生活保護受給者 | 15,000円（個人） |
| 世帯全員が住民税非課税で、年金収入80万円以下等 | 15,000円（個人）/ 24,600円（世帯） |
| 世帯全員が住民税非課税 | 24,600円（世帯） |
| 住民税課税〜課税所得380万円未満 | 44,400円（世帯） |
| 課税所得380万円以上690万円未満 | 93,000円（世帯） |
| 課税所得690万円以上 | 140,100円（世帯） |

**申請方法**: 市区町村の介護保険課に申請する。初回申請後は自動的に払い戻しが続く自治体が多い。該当する場合は市区町村から通知が届くことが多いが、届かない場合は自分から問い合わせること。

### 3-4. 高額医療・高額介護合算療養費制度

> **▶ 提示条件**: 医療費と介護費の両方が発生しているケース、または年間の合計負担が大きいと推測される場合に提示する。

同じ世帯で、1年間（毎年8月〜翌年7月）の医療保険と介護保険の自己負担を合算して、上限額を超えた分が払い戻される制度である。

#### 70歳以上の場合

| 区分 | 年間上限額 |
|---|---|
| 住民税非課税（低所得I） | 19万円 |
| 住民税非課税（低所得II） | 31万円 |
| 一般（課税所得145万円未満） | 56万円 |
| 現役並みI（課税所得145万円以上） | 67万円 |
| 現役並みII（課税所得380万円以上） | 141万円 |
| 現役並みIII（課税所得690万円以上） | 212万円 |

#### 70歳未満の場合

| 区分 | 年間上限額 |
|---|---|
| 住民税非課税 | 34万円 |
| 〜年間所得210万円以下 | 60万円 |
| 年間所得210万円超〜600万円以下 | 67万円 |
| 年間所得600万円超〜901万円以下 | 141万円 |
| 年間所得901万円超 | 212万円 |

**申請方法**: 加入している医療保険と市区町村の介護保険課の両方に申請が必要である。基準日（毎年7月31日）に加入していた医療保険に申請する。

### 3-5. 医療費控除（介護サービスの対象範囲）

> **▶ 提示条件**: 確定申告や税金の還付に関する質問があった場合、または介護費用の領収書を保管しているか確認する流れの中で提示する。

確定申告で医療費控除の対象となる介護サービスがある。

#### 対象となる介護サービス（主なもの）

- **施設サービス**: 特養の自己負担額の2分の1、老健・介護医療院の自己負担額の全額
- **居宅サービス（医療系）**: 訪問看護、訪問リハビリテーション、居宅療養管理指導、通所リハビリテーション、短期入所療養介護
- **居宅サービス（福祉系・上記医療系と併用する場合に限り対象）**: 訪問介護（生活援助中心型を除く）、通所介護、短期入所生活介護、夜間対応型訪問介護、小規模多機能型居宅介護など
- **おむつ代**: 医師が発行する「おむつ使用証明書」があれば対象。2年目以降は市区町村が発行する確認書で代替可能な場合あり

#### 控除額の計算

- 年間の医療費合計 - 保険金等で補填された額 - 10万円（または総所得金額の5%のいずれか低い方）= 医療費控除額
- 上限は200万円

### 3-6. 障害者控除対象者認定

> **▶ 提示条件**: 要介護認定を受けている親がおり、税負担の軽減策を探している場合に提示する。制度の存在自体を知らないケースが多いため、積極的に案内する。

要介護認定を受けている高齢者は、市区町村に申請することで「障害者控除対象者認定書」を取得できる場合がある。

- **対象**: 65歳以上で要介護1以上の認定を受けている方（自治体により基準が異なる）
- **控除額**: 障害者控除27万円、特別障害者控除40万円（所得税の場合）
- **効果**: 本人または扶養している家族の所得税・住民税が軽減される
- **注意**: 身体障害者手帳を持っていなくても申請できる。自治体によって認定基準が異なるため、まずは市区町村の窓口に確認すること

### 3-7. 世帯分離のメリット・デメリット・手続き

> **▶ 提示条件**: 相談者が世帯分離について質問した場合、または同居の親の介護費用が世帯合算で高額になっているケースで提示する。

親と同居している場合、住民票上の世帯を分けることで介護費用の負担が軽減される場合がある。

#### メリット

- 高額介護サービス費の上限額が下がる可能性がある（世帯の所得で判定されるため）
- 施設入所時の食費・居住費の減額（特定入所者介護サービス費）を受けられる可能性がある
- 介護保険料が下がる場合がある

#### デメリット

- 国民健康保険に加入している場合、世帯を分けると保険料が合計で増える場合がある
- 会社の健康保険の扶養から外れる可能性がある
- 各種届出で「世帯全員の住民票」が必要な場合に手続きが煩雑になる

#### 手続き

- 市区町村の窓口で「世帯変更届」を提出する
- 届出に必要なもの: 届出人の本人確認書類、印鑑（自治体による）
- 届出理由を聞かれることがあるが、「生計を別にしているため」で問題ない

#### 判断のポイント

世帯分離が有利かどうかは個々の状況によって異なる。以下を確認したうえで判断する。

1. 現在の介護保険の自己負担段階
2. 国民健康保険への影響
3. 会社の健康保険の扶養への影響
4. 税金の扶養控除への影響

**相談者には「得になるケースも損になるケースもありますので、具体的な数字を持って市区町村の窓口やファイナンシャルプランナーに相談されることをおすすめします」と案内する。**

### 3-8. 年金の種類と受給額の目安

> **▶ 提示条件**: 親の年金額が話題に上った場合、または介護費用と年金収入のバランスを検討する流れの中で提示する。

親の年金額は介護費用を考えるうえで重要な情報である。まず大まかな仕組みを理解してもらう。

| 年金の種類 | 受給額の目安（月額） | 対象者 |
|---|---|---|
| 国民年金（老齢基礎年金） | 満額で約6.8万円 | 自営業、専業主婦（夫）など |
| 厚生年金（老齢厚生年金）+ 基礎年金 | 約14万〜16万円 | 会社員・公務員として勤務した方 |

※受給額は加入期間や報酬額によって大きく異なる。上記はあくまで一般的な目安である。

#### 年金額の確認方法

- **年金振込通知書**: 毎年6月頃に届く（金額変更がない年は届かない場合あり）
- **ねんきんネット**: 日本年金機構のWebサービスで過去の記録や見込み額を確認できる
- **年金事務所**: 窓口で照会可能（委任状があれば家族でも確認できる）
- **年金額改定通知書**: 金額が変わった際に届く

### 3-9. 生活保護制度の概要（介護との関係）

> **▶ 提示条件**: 親の年金・貯蓄では介護費用を賄えない見通しの場合、または相談者が「お金が足りない」と明確に訴えた場合に提示する。提示時は丁寧な導入を心がけること。

年金や貯蓄だけでは介護費用を賄えない場合の最後のセーフティネットである。

- **要件**: 資産の活用、能力の活用、他の制度の活用、扶養義務者の援助を行ったうえで、なお生活に困窮する場合に受給できる
- **介護扶助**: 生活保護受給者は介護保険の自己負担分が介護扶助として支給される
- **注意点**: 預貯金や不動産などの資産がある場合は原則として活用が求められる。ただし住んでいる持ち家は、一定の条件のもとで保有が認められる場合がある
- **相談先**: 市区町村の福祉事務所（生活保護の担当窓口）

**相談者に伝えるときの配慮**: 「生活保護」という言葉に抵抗感を持つ方も多い。「生活を支えるための公的な制度のひとつです」と伝え、利用すること自体は権利であることを丁寧に説明する。

### 3-10. 介護費用の確定申告での控除

> **▶ 提示条件**: 確定申告の時期（1〜3月頃）や、相談者が税金還付の可能性について質問した場合に提示する。

介護にかかった費用の一部は、確定申告で税金の還付を受けられる場合がある。

- **医療費控除**: 3-5で記載した対象サービスの自己負担額を医療費控除として申告する
- **社会保険料控除**: 親の介護保険料を子が支払った場合、子の社会保険料控除に含められる
- **扶養控除**: 親の合計所得金額が48万円以下（年金収入のみの場合158万円以下）であれば扶養控除の対象になる可能性がある
- **障害者控除**: 3-6の認定を受けていれば障害者控除を適用できる

### 3-11. 特定入所者介護サービス費（補足給付）の預貯金要件

> **▶ 提示条件**: 施設入所を検討中で食費・居住費の減額制度について質問があった場合、または3-1の施設費用に対して経済的不安が示された場合に提示する。

特定入所者介護サービス費（補足給付）は、住民税非課税世帯に属する施設入所者の食費・居住費を軽減する制度である。ただし、預貯金等の資産要件が段階ごとに設定されている。

| 利用者負担段階 | 対象者の概要 | 預貯金等の基準（単身/夫婦） |
|---|---|---|
| 第1段階 | 生活保護受給者、老齢福祉年金受給者で住民税非課税世帯 | 1,000万円以下 / 2,000万円以下 |
| 第2段階 | 住民税非課税世帯で、年金収入等80万円以下 | 650万円以下 / 1,650万円以下 |
| 第3段階(1) | 住民税非課税世帯で、年金収入等80万円超120万円以下 | 550万円以下 / 1,550万円以下 |
| 第3段階(2) | 住民税非課税世帯で、年金収入等120万円超 | 500万円以下 / 1,500万円以下 |

※預貯金等には、預貯金のほか有価証券等も含まれる。申請時に通帳の写し等の提出が必要である。
