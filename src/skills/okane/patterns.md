## 4. よくある状況と対応パターン

### 4-1. 「親の年金額がわからない」

**背景**: 親がお金の話をしたがらない、離れて暮らしていて把握できていない、というケースは非常に多い。

**対応の流れ**:

1. まず、年金額がわからなくても不自然なことではないと伝えて安心してもらう
2. 確認方法を案内する
   - 親の自宅に届いている「年金振込通知書」や「年金額改定通知書」を確認
   - 通帳の振込記録から逆算する（年金は偶数月に2か月分が振り込まれる）
   - 親本人が「ねんきんネット」に登録していれば、そこで確認
   - 年金事務所に本人が行く、または委任状を持って家族が照会する
3. 年金額がわかったら、介護費用との差額を確認し、不足分をどう補うかを一緒に考える

### 4-2. 「介護費用を兄弟で分担する方法」

**背景**: 兄弟間での費用分担はトラブルになりやすく、感情的な対立に発展することも多い。

**対応の流れ**:

1. 費用分担に正解はないことを前提として共有する
2. よく使われる分担の考え方を紹介する
   - **均等割り**: 兄弟の人数で均等に分ける
   - **収入比率**: それぞれの収入に応じて割合を決める
   - **役割分担**: 近くに住む人が「時間」を、遠方の人が「お金」を出す
   - **基金方式**: 毎月一定額を共通口座に積み立て、そこから支出する
3. 話し合いの場を設けることを提案する（できれば全員が集まれるタイミングで）
4. 話し合いの前に、現在の介護費用の総額を整理しておくことを勧める
5. 合意内容は簡単でもよいので文書にしておくことを推奨する

### 4-3. 「親の貯蓄が底をつきそう」

**背景**: 切迫した不安を抱えている。冷静に使える制度を整理することが最優先である。

**対応の流れ**:

1. 不安な気持ちを受け止めたうえで、使える制度を優先順位をつけて案内する
2. 制度活用の優先順位

| 優先度 | 制度 | 効果 |
|---|---|---|
| 1 | 高額介護サービス費の申請 | 月々の自己負担に上限が設けられる |
| 2 | 特定入所者介護サービス費（施設入所の場合） | 食費・居住費が大幅に減額される |
| 3 | 世帯分離の検討 | 上記の制度で有利な区分になる可能性 |
| 4 | 障害者控除対象者認定の取得 | 税負担の軽減 |
| 5 | 社会福祉協議会の生活福祉資金貸付 | 一時的な資金不足を補う低金利の貸付 |
| 6 | 生活保護の相談 | 最終的なセーフティネット |

3. 介護サービスの見直し（ケアマネジャーに相談して、過不足を調整する）も並行して提案する

### 4-4. 「自分の老後資金も心配」

**背景**: 親の介護費用を負担することで、自分自身の将来に不安を感じている。

**対応の流れ**:

1. 「自分の将来も大切にしたい」という気持ちは正当であると伝える
2. 親の介護費用はまず親の年金・資産から賄うのが基本であることを確認する
3. 公的制度を最大限活用して、家族の持ち出しを最小限にする方針を整理する
4. 自分の負担が重くなりすぎないよう、限度額を決めておくことを勧める
5. 必要に応じてファイナンシャルプランナーへの相談を提案する

### 4-5. 「世帯分離した方がいいと聞いたが」

**背景**: インターネット等で情報を得て相談に来るケースが増えている。

**対応の流れ**:

1. 世帯分離のメリット・デメリットを3-7に基づいて説明する
2. 以下の判断ポイントを確認する
   - 現在の介護保険の自己負担割合と高額介護サービス費の上限額
   - 国民健康保険に加入しているか、会社の健康保険の扶養に入っているか
   - 税金上の扶養控除を利用しているか
3. 個別の試算が必要であるため、市区町村の窓口またはファイナンシャルプランナーに相談することを勧める
4. 「世帯分離は万能ではなく、状況によっては損になることもあります」と正確に伝える
